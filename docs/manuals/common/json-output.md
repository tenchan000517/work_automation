# JSON出力マニュアル

ヒアリングシートのデータをJSON形式で出力する手順

## JSON出力について

ヒアリングシートに入力されたデータをJSON形式で出力し、AIに渡すためのデータとして使用します。

**用途:**
- 構成案プロンプト → 外部AIに投げる
- Claude Code用HP作成指示文のデータ

**注意:**
- サーバー情報（Part③）は含まれません（セキュリティ対策）

## Step 1: メニューを開く

ヒアリングシートのスプレッドシートを開き、対象の企業シートを選択します。

**HP制作の場合:**
メニュー「**4.📝 構成案作成**」→「**📤 HP制作用JSON出力**」

<img src="/images/hp/menu-json-output.png" alt="JSON出力メニュー" style="max-width: 300px; border: 1px solid #ddd; border-radius: 4px; margin: 16px 0;" />

## Step 2: 対象企業を選択

ダイアログが開いたら、ドロップダウンから対象企業を選択します。

<img src="/images/hp/dialog-json-output.png" alt="JSON出力ダイアログ" style="max-width: 100%; border: 1px solid #ddd; border-radius: 4px; margin: 16px 0;" />

## Step 3: JSONを生成

「**🔄 生成・保存**」ボタンをクリックします。

→ JSON形式のデータがテキストエリアに表示されます
→ Part④に自動保存されます

**出力されるJSONの構造:**
```json
{
  "企業名": "株式会社○○",
  "更新日時": "2026/1/30 15:30:00",
  "Part①_基本情報": {
    "会社の詳細情報": {
      "設備・施設": "...",
      "オフィス・店舗情報": "...",
      "代表メッセージ": "...",
      ...
    },
    "担当者情報_企業情報": {
      "従業員数": "...",
      "代表者名": "...",
      ...
    }
  },
  "Part②_ヒアリング情報": {
    "ゴール・コンバージョン": { ... },
    "ターゲットの深掘り": { ... },
    "強みの深掘り": { ... },
    "表現の方向性": { ... },
    "SEO・キーワード設計": { ... },
    "新規作成の確認": { ... }
  }
}
```

## Step 4: JSONをコピー

「**📋 コピー**」ボタンをクリックしてJSONをコピーします。

→ クリップボードにコピーされます

## Step 5: AIに貼り付け

コピーしたJSONを構成案プロンプトや指示文に貼り付けて使用します。

---

## JSONの活用方法

### 用途1: 構成案プロンプト

1. JSON出力でデータをコピー
2. 「📋 構成案プロンプト生成」でプロンプトを生成
3. プロンプトにJSONが自動で埋め込まれる
4. Claude/ChatGPTに貼り付けて構成案を作成

### 用途2: Claude Code指示文

1. JSON出力でデータをコピー
2. 「🤖 Claude Code指示文生成」で指示文を生成
3. 指示文にJSONが自動で埋め込まれる
4. Claude Codeに貼り付けてセットアップ

## トラブルシューティング

### JSONが空になる

**原因:**
- ヒアリングシートにデータが入力されていない

**解決策:**
- Part①（基本情報）が入力されているか確認
- Part②（ヒアリング情報）が入力されているか確認

### 特定の項目が含まれない

**原因:**
- 該当のセルが空白

**解決策:**
- ヒアリングシートの該当セルにデータを入力

### Part③が含まれない

**仕様です:**
- サーバー情報（Part③）はセキュリティ上の理由で意図的に除外されています
- 必要な場合は手動でコピー＆貼り付け
