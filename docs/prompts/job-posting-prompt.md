## 【出力指示】

### A. 求人原稿

以下の構成で作成してください：

#### 1. 職務内容（具体的な業務内容）

**構成：**
- 冒頭：会社紹介（設立年、事業内容、理念など）
- 中盤：「あなたにお任せするのは以下業務です！」→ 業務をStep形式（3〜5ステップ）で記載
- 補足：未経験者への安心感を伝えるフォロー文
- 後半：「＼＼ 当社のポイント //／」で3つの魅力を◇で紹介
  - 各ポイントはインタビュー内容から具体的なエピソードを引用

#### 2. 求人概要
- 募集職種 / 雇用形態 / 試用期間 / 想定年収・月給 / 勤務地 / 募集人数 / 募集背景

#### 3. 応募要件
- 必須スキル / 歓迎スキル / 求める人物像 / 必要資格

#### 4. 待遇・福利厚生
- 給与詳細 / 勤務時間 / 休日休暇 / 福利厚生一覧

---

### B. サムネイル用キャッチコピー（5枚分）

**条件：**
- 1つ15文字以内推奨（最大20文字）
- ターゲット層に刺さる言葉
- 入力素材から印象的なフレーズを抽出・活用

**5つのテーマ：**
1. 働きやすさ・ワークライフバランス
2. 成長・キャリア
3. 職場の雰囲気・人間関係
4. 待遇・福利厚生
5. 会社の強み・安定性

---

## 【文体・トーンの指示】

- 親しみやすく明るい文体
- 絵文字・記号を適度に使用（例：＾＾★ ♪ ◇ など）
- 「！」で活気を出す
- 堅すぎずカジュアルすぎないバランス
- 具体的な数字や事実を盛り込む
- インタビューの生の声を「」で引用して信頼性を高める

**避けること：**
- 汎用的・抽象的な表現（「アットホームな職場」「風通しの良い環境」など具体性のない言葉）
- 入力素材にない情報の創作
- 誇張表現
