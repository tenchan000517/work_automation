# HP作成マニュアル

## 概要
Claude Codeを使用して、テンプレートからHPを生成する。

## 担当者
- 河合

## 使用ツール
- Claude Code
- VSCode

## 関連リソース
- HPテンプレート: https://sing-hp-template.vercel.app/
- 修正マニュアル: https://sing-hp-template.vercel.app/manual

## フロー

### 1. テンプレート準備
HPテンプレートを準備する。
- テンプレートリポジトリをクローン
- 必要な設定を確認

### 2. Claude Code実行
原稿を元にHPを生成する。
- Claude Codeを起動
- 原稿とテンプレートを元にHP生成を指示

### 3. 動作確認
表示崩れ・リンク切れをチェックする。
- 各ページの表示確認
- リンクの動作確認
- レスポンシブ対応の確認

### 4. ステータス更新
ステータスを更新して次の工程へ。

## チェックポイント
- テンプレートからHPが正しく生成されているか
- 表示崩れがないか
- リンク切れがないか

## 次の担当者
