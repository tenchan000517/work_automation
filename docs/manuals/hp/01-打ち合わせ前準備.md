# 打ち合わせ前準備マニュアル

## 目的

初回打ち合わせをスムーズに行うための事前準備。
ヒアリングシートを作成し、撮影可能日程も事前に確認しておく。

## 担当者

- 河合

## フロー

### 1. 回答確認

フォームの事前回答を確認する。

回答があれば次のステップでヒアリングシートを作成します。
回答がない場合は、先方にリマインドするか「新規作成（手動）」で空のシートを作成します。

**リンク:**
- [ヒアリングシート](https://docs.google.com/spreadsheets/d/1GO5fyOd-0lT_OMpLNw6rIZDRA6jrK4lzIAt-0tDxGvc/)

### 2. シート作成

ヒアリングシートのスプレッドシートを開き、メニューからシートを作成する。

#### パターンA：フォーム回答済みの場合

1. ヒアリングシートのスプレッドシートを開く
2. メニュー「ヒアリングシート」をクリック
3. 「新規作成（フォーム回答から）」を選択
4. 企業一覧から該当企業を選択
5. 「実行」をクリック

→ 「○○株式会社」シート（タブ）が自動作成される
→ フォームの回答データが自動転記される

![メニューを開く](/images/gas-menu.png)
*STEP 1: メニュー「📋ヒアリングシート」を開く*

![フォーム回答から選択](/images/gas-form-select-dialog.png)
*STEP 2: パターンA - フォーム回答から企業を選択*

#### パターンB：フォーム未回答の場合

1. ヒアリングシートのスプレッドシートを開く
2. メニュー「ヒアリングシート」をクリック
3. 「新規作成（手動）」を選択
4. 企業名を入力して「OK」をクリック

→ 「○○株式会社」シート（タブ）が作成される（空の状態）
→ 初回打ち合わせで直接ヒアリングしながら入力

![手動作成](/images/gas-manual-create.png)
*STEP 2: パターンB - 手動で企業名を入力*

#### ⚠️ 企業名入力ルール（重要）

手動作成時の企業名は必ず以下のルールで入力：

✅ **正しい例：**
- 株式会社○○
- ○○株式会社
- 有限会社○○

❌ **間違い例：**
- (株)○○　← カッコ省略NG
- ㈱○○　← 省略記号NG
- ○○ 株式会社　← 余計なスペースNG

※ルール通りでないと機能が正しく動作しません

#### 作成後の確認

- 企業名が正しく入っているか
- （パターンAの場合）Part①の黄色セルにデータがあるか

### 3. フォルダ作成

ヒアリングシートのメニューから企業フォルダを作成する。

**手順:**
1. ヒアリングシートを開く
2. メニュー「📁 フォルダ」→「📂 企業シートから作成」
3. 対象の企業シートを選択
4. 「📁 フォルダを作成」をクリック

→ 企業フォルダが自動作成され、URLが企業シートに保存されます

![フォルダ作成メニュー](/images/gas-folder-menu.png)
*STEP 2: メニュー「📁 フォルダ」→「📂 企業シートから作成」*

![フォルダ作成完了](/images/gas-folder-complete.png)
*STEP 4: フォルダ作成完了*

**作成されるフォルダ:**
```
HP・LP フォルダ/
└── [企業名]/ ← 今回作成
    └── （文字起こし原稿などを格納）
```

**補足:**
- URLは企業シートのPart③に自動保存されます
- 素材フォルダ（ページ別サブフォルダ）はNo.6（素材撮影）でテンプレート確定後に作成します

**リンク:**
- [HP・LPフォルダ](https://drive.google.com/drive/folders/1Zi2zn57JA3wZQvrEUwGN26jZkRDodWe-)
- [ヒアリングシート](https://docs.google.com/spreadsheets/d/1GO5fyOd-0lT_OMpLNw6rIZDRA6jrK4lzIAt-0tDxGvc/)

### 4. 撮影日確認

撮影担当者（川崎）の空き日程を確認する。

初回打ち合わせで先方に撮影候補日を提示できるよう、撮影担当者に事前確認します。

**確認内容:**
撮影可能な日程を5候補程度聞いておく

**送信手順:**
1. 連絡フォーマットをコピー
2. 企業名・日程を入力
3. ワークスに貼り付けて送信

**撮影日程確認フォーマット:**
```
@川崎さん
お疲れ様です。

下記案件の撮影日程について、ご都合を確認させてください。

━━━━━━━━━━━━━━━━━━
■ 案件情報
━━━━━━━━━━━━━━━━━━
【企業名】
【初回打ち合わせ予定日】
【撮影希望時期】

━━━━━━━━━━━━━━━━━━
■ 確認事項
━━━━━━━━━━━━━━━━━━
上記期間で撮影可能な日程を5候補程度教えてください。

よろしくお願いいたします。
```

### 5. リマインド

打ち合わせ参加者にリマインドを送信する。

初回打ち合わせの参加者（自分を除く）にリマインドを送ります。

**送信手順:**
1. 連絡フォーマットをコピー
2. 企業名・日時・Meet URLを入力
3. ワークスに貼り付けて送信

**リマインドフォーマット:**
```
お疲れ様です。

明日の打ち合わせについてリマインドです。

━━━━━━━━━━━━━━━━━━
■ 打ち合わせ概要
━━━━━━━━━━━━━━━━━━
【企業名】
【日時】
【場所】Google Meet

━━━━━━━━━━━━━━━━━━
■ Meet URL
━━━━━━━━━━━━━━━━━━
（ここにURLを貼る）

よろしくお願いいたします。
```

### 6. リアクション確認

メンバーの確認状況をチェックする。

**ワークスのスタンプルール:**

| ルール | 説明 |
|--------|------|
| ①完了していないならスタンプを押さない | スタンプ = 完了の意思表示 |
| ②5分以上かかるものはタスク・カレンダーに落とし込んでからスタンプ | すぐ対応できないものは先にタスク化 |
| ③既読したらOKスタンプ → 完了したらグッドスタンプに変更 | 👌（OK）= 確認した、対応中 / 👍（グッド）= 完了した |

撮影可能日5候補が確定したら、次のステップへ進みます。

### 7. 録画準備

NOTTAの録画設定を確認する。

初回打ち合わせ前に、NOTTAの録画要領を確認しておきます。

**事前確認チェックリスト:**
- [ ] NOTTAにログインできるか
- [ ] Web会議の自動文字起こし機能を確認
- [ ] Google Meetとの連携方法を確認

### 8. ステータス更新

ステータスを更新して次の工程へ。

**手順:**
1. ヒアリングシートを開く
2. ヒアリングシートの任意のメニューから「✏️ ステータス更新」を選択
3. 現在タスクを次の業務に変更
4. タスク保持者を次の担当者に変更
5. 状態を「次の担当へ」に変更
6. メモに必要な申し送り事項を入力
7. 「更新」をクリック

**リンク:**
- [ヒアリングシート](https://docs.google.com/spreadsheets/d/1GO5fyOd-0lT_OMpLNw6rIZDRA6jrK4lzIAt-0tDxGvc/)

## 成果物

- ヒアリングシート（企業名入り、フォームデータ転記済み）
- 企業フォルダ（作成済み）
- 撮影可能日程（5候補程度）

## 備考

- フォーム回答が来ていない場合は「新規作成（手動）」で空のシートを作成
- 撮影日程は初回打ち合わせで先方と最終調整
- 素材フォルダはNo.6（素材撮影）でテンプレート確定後に作成
