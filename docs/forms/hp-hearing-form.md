## セクション3: サーバー情報

### セクション冒頭の説明文（★強調）

```
【重要】このセクションは「把握状況の確認」です

ログイン情報やパスワードなどの入力は不要です。
「把握している」「把握していない」「わからない」のいずれかをお選びください。

把握していない項目があっても問題ありません。
打ち合わせ時に一緒に確認・整理していきます。
```

### 3-1. サーバー管理の希望（★分岐の起点）

| 選択肢 | 説明 | 追加質問 |
|--------|------|---------|
| A. Singに移管する | ドメイン・サーバーをSingで管理 | 共通 → ABD → メール → 完了 |
| B. メール契約だけ現行のまま残す | サーバー移管、メールは現行維持 | 共通 → ABD → 完了 |
| C. 自社で管理する | 納品物をお渡しし、御社でアップロード | 共通 → C → 完了 |
| D. 検討中 | 打ち合わせで相談したい | 共通 → ABD → メール → 完了 |

## ページ遷移（条件分岐）

```
セクション1: 基本情報
    ↓
セクション2: HP制作の方向性
    ↓
セクション3: サーバー情報
    ↓
3-1. サーバー管理の希望
    ├─ A/B/C/D → 共通質問（3-2〜3-4）
    │
    ├─ A/B/D → ABD追加質問（3-5〜3-11）
    │   ├─ A/D → メール関連（3-12〜3-14） → 完了
    │   └─ B → 完了
    │
    └─ C → C追加質問（3-15〜3-16） → 完了
```

※Googleフォームの制約により、完全な動的分岐は困難。
全質問を表示し、不要な質問は任意とする形で対応。

## 要注意パターン（打ち合わせで重点確認）

| パターン | 対応 |
|---------|------|
| ログイン情報がない | 外部委託先に連絡 or 再発行手続き |
| AuthCode取得方法がわからない | 打ち合わせで手順説明 |
| 同ドメインでメール使用中 + A選択 | メール移行計画が必須 |
| 過去メール保持希望 | IMAP移行の説明 |
| 外部連絡不可 | 移管困難、要相談 |
| 検討中 | 全パターン想定で情報収集 |
