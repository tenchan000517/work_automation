## テンプレート概要

**テンプレート名**: LeadGen Minimal

**型（Type）**: Lead Gen（CV特化）
**表現（Design）**: Minimal Pro（洗練ミニマル）

**コンセプト**:
高単価サービスのコンバージョンに特化。**余白を贅沢に使い、高級感と信頼感を演出**しながら、**最短でCVへ導く**。
情報を削ぎ落とし、**本当に必要なことだけを伝える**。
「この会社はプロフェッショナルだ」と感じさせるデザイン。

**想定クライアント**:
- BtoB企業（コンサルティング、システム開発、専門サービス）
- 高単価サービス提供企業
- 問い合わせ獲得が最優先の企業
- 「安売り競争から脱却したい」企業

**想定ユーザー（閲覧者）**:
- 企業の意思決定者（40-60代）: 「信頼できるパートナーを探している」
- 担当者（30-50代）: 「上司に提案するための情報を集めている」
- 比較検討中のユーザー: 「複数社を比較している」

**このテンプレートで達成すべきこと**:
1. 問い合わせフォームへの到達率向上
2. フォーム完了率の向上（離脱防止）
3. 「高単価でも依頼したい」という信頼感
4. 価格競争からの脱却

## 技術的制約（これらは変更不可）

1. **ベース**: template-fullorder からクローン
2. **データ管理**: site.json + 各ページ上部のコンテンツデータ定数
3. **スタイル**: Tailwind CSS 4.x（globals.css の @theme 内ブランドカラーのみ変更可）
4. **アニメーション**: Framer Motion 12.x
5. **画像**: next/image コンポーネント使用
6. **レスポンシブ**: PC優先、SP対応必須

### PART 1: 全体戦略（→ 01-strategy.md に保存）

#### 1-1. このテンプレートの独自価値

**👤 専門家①（マーケティング）の視点**:
- Standard テンプレートとの決定的な違いは何か
- このテンプレートでしか実現できないコンバージョン戦略は何か
- 「3クリック以内にCV」をどう実現するか

**👤 専門家②（心理学）の視点**:
- このテンプレートが活用する心理的メカニズムは何か
- 高単価サービスへの信頼をどう構築するか
- 「余白」が心理に与える影響は何か

**👤 専門家③（デザイン）の視点**:
- このテンプレートのビジュアルアイデンティティは何か
- 「Minimal」表現をどう具体化するか
- AIデザインとの差別化ポイントは何か

#### 1-2. カラー戦略

- メインカラーの推奨系統と心理効果
- アクセントカラーの使いどころ（CTAのみに限定）
- 背景色のバリエーション（白、オフホワイト、グレー）
- **避けるべき色の組み合わせ**

#### 1-3. タイポグラフィ戦略

- 見出しのサイズ階層（H1, H2, H3）- 大胆な強弱
- 本文とのコントラスト
- 日本語特有の考慮点（字間、行間）
- **「Minimal」表現におけるタイポグラフィの役割**

#### 1-4. 余白とリズム戦略

- セクション間の余白の考え方（余白率40%以上）
- 「Minimal」表現における余白の意味
- 密と疎のリズム設計
- **均一な余白を避ける具体的な方法**

#### 1-5. アニメーション戦略

- どこにアニメーションを使い、どこに使わないか
- 「Minimal」表現におけるアニメーションの役割（控えめに）
- 過剰にならない基準
- 具体的なアニメーションパターン（2種類程度に限定）

## 禁止事項

1. **汎用的な表現の禁止**
   - ❌「フルワイドのヒーローセクション」
   - ✅「画面上部40%を余白とし、中央にキャッチコピーのみ。72pxの太字。下部にスクロール誘導の矢印」

2. **理由なき設計の禁止**
   - ❌「CTAボタンを配置」
   - ✅「各セクション末尾にCTAを配置。理由: ユーザーが納得したタイミングでアクションできるように」

3. **AIデザインの典型パターン禁止**
   - ❌ 全要素にカードコンテナ
   - ❌ 紫系グラデーション
   - ❌ 全セクションで同じ余白
   - ❌ 全要素にフェードインアニメーション

4. **曖昧な指示の禁止**
   - ❌「適切な余白を設ける」
   - ✅「上余白160px、下余白80px。上を大きくする理由: 余白で高級感を演出し、コンテンツに希少性を持たせるため」
