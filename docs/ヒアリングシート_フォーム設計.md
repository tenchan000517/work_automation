# 先方提出用ヒアリングシート（Googleフォーム）設計書

## 概要

**目的:** 先方（クライアント企業）に入力してもらうヒアリングシート
**形式:** Googleフォーム
**連携:** 回答データをGASで内部用ヒアリングシートに自動転記
**対象:** Part①（基本情報）のみ ※Part②は打ち合わせ時に内部で記入

## 内部シートとの対応表

| フォームセクション | 内部シートセクション |
|-------------------|---------------------|
| 1. 企業概要 | 企業概要 |
| 2. 雇用形態・職種 | 求人区分・雇用形態 |
| 3. 勤務条件 | 勤務時間 + 休憩時間 + 残業 + 休日 |
| 4. 給与・待遇 | 給与 + 待遇・福利厚生 |
| 5. 業務内容 | 作業内容 + 製品・商品・部品 + その他 |

※Part②（会社紹介、社員の声、募集背景等）は打ち合わせ時に内部で記入

---

## 運用フロー

```
1. 営業が受注
   ↓
2. 先方にGoogleフォームのURLを送付
   ↓
3. 先方がフォームに入力・送信
   ↓
4. 回答がスプレッドシート「フォーム回答」シートに自動集約
   ↓
5. GASで「転記」ボタンを押す
   ↓
6. 内部用ヒアリングシートに新しいシートが作成され、データが転記される
   ↓
7. 担当者が内部シートを確認・補足入力（Part②）
   ↓
8. AI用データ出力（既存のGASスクリプト）
```
