# 👤 専門家① WEBマーケティングのプロ（30年の経験）の提案

## 提案するテンプレート一覧

| テンプレート名 | ページ数 | 特徴 | 最適なクライアント |
|--------------|---------|------|------------------|
| Lead Gen（リード獲得特化） | 5ページ | LP型トップ+最短CVルート | BtoB企業、問い合わせ獲得が最優先の企業 |
| Recruit Pro（採用特化） | 8ページ | 求職者ジャーニー完全対応 | 採用難の中小企業、人材確保が急務の企業 |
| Local Business（地域密着） | 6ページ | MEO対策・地域キーワード最適化 | 建設業、製造業、士業など地域商圏ビジネス |
| Authority（権威構築） | 9ページ | 実績・事例を前面に出す設計 | 専門性で勝負したい企業、高単価サービス |

## 各テンプレートの詳細

### テンプレート名: Lead Gen（リード獲得特化）
- **コンセプト**: Standard は情報網羅型だが、CVまでの導線が弱い。ABテストで証明された「3クリック以内にCV」を実現
- **ページ構成**: TOP（LP型）/ サービス / 事例・実績 / お問い合わせ / プライバシー
- **特徴的な要素**:
  - TOPページにCTAを最低3箇所配置
  - 追従するフローティングCTAボタン
  - フォーム到達前にマイクロコンバージョン（資料DL等）設置
- **想定クライアント**: 「問い合わせを増やしたい」が第一目標の企業
- **期待される効果**: 標準テンプレート比 CVR 1.5〜2倍

### テンプレート名: Recruit Pro（採用特化）
- **コンセプト**: 採用サイトは企業サイトとは別の論理が必要。求職者の意思決定プロセスに完全対応
- **ページ構成**: TOP / 会社を知る / 仕事を知る / 社員を知る / 数字で見る / 募集要項 / エントリー / FAQ
- **特徴的な要素**:
  - 社員インタビューテンプレート（写真+テキスト最適配置）
  - 1日の流れタイムライン
  - 福利厚生ビジュアル一覧
  - IndeedやWantedly連携を想定した構造化データ
- **想定クライアント**: 人材不足が経営課題の中小企業
- **期待される効果**: 応募数増加、ミスマッチ低減

### テンプレート名: Local Business（地域密着）
- **コンセプト**: 「地域名+業種」で検索する見込み客を確実に獲得
- **ページ構成**: TOP / サービス / 対応エリア / 施工事例（実績） / 会社概要 / お問い合わせ
- **特徴的な要素**:
  - Googleマップ埋め込み最適化
  - 地域名を自然に含むテキストテンプレート
  - 施工事例のBefore/After表示機能
  - お客様の声セクション（構造化データ対応）
- **想定クライアント**: 建設業、リフォーム、製造業、士業
- **期待される効果**: 地域検索での上位表示、MEO効果向上

### テンプレート名: Authority（権威構築）
- **コンセプト**: 高単価・専門サービスは「なぜこの会社か」の説得が必要
- **ページ構成**: TOP / 選ばれる理由 / サービス / 事例詳細 / お客様の声 / 代表メッセージ / 会社概要 / コラム / お問い合わせ
- **特徴的な要素**:
  - 数字で見せる実績セクション（カウントアップアニメーション）
  - 詳細な事例ページテンプレート（課題→解決→成果）
  - メディア掲載・受賞歴セクション
  - 専門コラム機能
- **想定クライアント**: コンサル、設計事務所、高単価サービス業
- **期待される効果**: 価格競争からの脱却、指名検索増加

# 👤 専門家② 行動心理学の権威の提案

## 提案するテンプレート一覧

| テンプレート名 | ページ数 | 特徴 | 最適なクライアント |
|--------------|---------|------|------------------|
| Trust Builder（信頼構築型） | 7ページ | 社会的証明と認知負荷低減を最大化 | 新規参入、知名度の低い企業 |
| Story（ナラティブ型） | 6ページ | 感情訴求で共感を生む構成 | 理念重視の企業、創業ストーリーが強い企業 |
| Logic Flow（論理説得型） | 7ページ | 段階的な説得フローを実装 | BtoB、検討期間の長い商材 |
| Quick Decision（即決促進型） | 4ページ | 認知負荷を極限まで下げる | 緊急性の高いサービス、明確なニーズ |

## 各テンプレートの詳細

### テンプレート名: Trust Builder（信頼構築型）
- **コンセプト**: 日本人は「みんなが選んでいる」「専門家が認めている」に弱い。社会的証明を全面展開
- **ページ構成**: TOP / 選ばれる理由 / 実績・数字 / お客様の声 / 会社概要 / サービス / お問い合わせ
- **特徴的な要素**:
  - ファーストビュー直下に「〇〇社導入」「顧客満足度〇％」
  - お客様の声を写真付きで複数配置（具体的な数字入り）
  - 取引先ロゴ一覧
  - 資格・認証バッジの視覚的表示
- **想定クライアント**: 知名度がまだ低い企業、新規事業
- **期待される効果**: 初訪問者の離脱率低減、信頼感醸成

### テンプレート名: Story（ナラティブ型）
- **コンセプト**: 人は論理より物語で動く。創業の想い、課題解決の旅を追体験させる
- **ページ構成**: TOP（物語導入）/ ストーリー / サービス / 事例 / チーム / お問い合わせ
- **特徴的な要素**:
  - スクロールで進む物語演出（パララックス活用）
  - 「Before → 転機 → After」の型
  - 創業者・代表の顔が見えるデザイン
  - 感情を動かす写真配置（人の表情重視）
- **想定クライアント**: 理念やビジョンが明確な企業、採用も重視
- **期待される効果**: 共感による強いブランド印象、記憶への定着

### テンプレート名: Logic Flow（論理説得型）
- **コンセプト**: BtoBや高額商材は「比較検討」が前提。段階的に疑問を解消する設計
- **ページ構成**: TOP / 課題提起 / 解決策（サービス）/ 比較・選ばれる理由 / 導入フロー / FAQ / お問い合わせ
- **特徴的な要素**:
  - 「こんなお悩みありませんか？」から始まるファーストビュー
  - 比較表（自社 vs 競合、ただし競合名は出さない）
  - ステップ形式の導入フロー図解
  - 詳細なFAQ（反論処理を事前に）
- **想定クライアント**: BtoB、システム導入、コンサルティング
- **期待される効果**: 検討段階の見込み客の説得、営業コスト削減

### テンプレート名: Quick Decision（即決促進型）
- **コンセプト**: 認知負荷を下げ、迷わせない。選択肢を減らし即行動を促す
- **ページ構成**: TOP（ほぼ完結）/ サービス詳細 / お問い合わせ / 会社概要
- **特徴的な要素**:
  - TOPページで完結する情報設計
  - 電話番号・LINEを大きく表示
  - 選択肢は最大3つまでに制限
  - 「今すぐ」「本日中」などの時間的プレッシャー要素
- **想定クライアント**: 水道修理、鍵屋、緊急対応サービス
- **期待される効果**: 即時コンバージョン、電話問い合わせ増

# 👤 専門家③ 人間最強のWEBデザイナーの提案

## 提案するテンプレート一覧

| テンプレート名 | ページ数 | 特徴 | 最適なクライアント |
|--------------|---------|------|------------------|
| Visual Impact（ビジュアル重視） | 5ページ | 写真・動画で魅せる、テキスト最小限 | 製造業（工場）、建築、飲食 |
| Minimal Pro（洗練ミニマル） | 6ページ | 余白と typography で勝負 | 設計事務所、デザイン会社、高級サービス |
| Dynamic Flow（ダイナミック） | 7ページ | アニメーションで体験を演出 | 先進的イメージを持たせたい企業 |
| Magazine（読ませる型） | 8ページ | 雑誌のようなエディトリアル | 採用重視、ブランディング重視 |

## 各テンプレートの詳細

### テンプレート名: Visual Impact（ビジュアル重視）
- **コンセプト**: 製造業や建築は「見せれば伝わる」。写真・動画が主役、テキストは添え物
- **ページ構成**: TOP（全画面ビジュアル）/ Works / About / Service / Contact
- **特徴的な要素**:
  - ファーストビュー全画面動画または写真スライダー
  - 工場・現場の臨場感ある写真レイアウト（グリッド崩し）
  - テキストは最小限、キャッチコピーのみ大きく
  - ホバーで動画再生するギャラリー
- **想定クライアント**: 製造業、建設業、飲食業
- **期待される効果**: 「この会社、しっかりしてそう」という第一印象

### テンプレート名: Minimal Pro（洗練ミニマル）
- **コンセプト**: AIが生成しがちな「詰め込みすぎ」の対極。余白が価値を生む
- **ページ構成**: TOP / Philosophy / Works / Service / Team / Contact
- **特徴的な要素**:
  - 余白率40%以上のレイアウト
  - フォントサイズの大胆な強弱（16px vs 72px）
  - モノトーン基調+アクセント1色のみ
  - 横書きと縦書きの混在（日本語の美しさ活用）
  - スクロールに応じた要素のフェードイン
- **想定クライアント**: 設計事務所、デザイン会社、高単価サービス
- **期待される効果**: 高級感、専門性の演出

### テンプレート名: Dynamic Flow（ダイナミック）
- **コンセプト**: 静的なサイトとの差別化。動きで「先進的」「革新的」を表現
- **ページ構成**: TOP / About / Service / Works / News / Recruit / Contact
- **特徴的な要素**:
  - スクロールトリガーのパララックスアニメーション
  - カーソル追従のインタラクション
  - セクション遷移時の大胆なトランジション
  - 数字のカウントアップ演出
  - ただし、読み込み速度は妥協しない（Framer Motion最適化済み）
- **想定クライアント**: IT企業、先進的イメージを持たせたい企業
- **期待される効果**: 競合との差別化、記憶に残るサイト体験

### テンプレート名: Magazine（読ませる型）
- **コンセプト**: 「読む」ことで会社を理解させる。エディトリアルデザインの応用
- **ページ構成**: TOP / Feature（特集）/ People / Culture / Work / Numbers / Entry / Contact
- **特徴的な要素**:
  - 雑誌のような見出しタイポグラフィ
  - 写真とテキストの有機的なレイアウト（段組み、回り込み）
  - 引用ブロック、プルクォートの活用
  - スクロールで読み進める長編コンテンツ対応
  - 目次ナビゲーション
- **想定クライアント**: 採用に力を入れる企業、ブランディング重視
- **期待される効果**: 滞在時間増加、深い理解と共感

# 最終統合提案：優先度順テンプレートリスト

| 優先度 | テンプレート名 | 推薦専門家 | 実装の理由 |
|-------|--------------|-----------|-----------|
| **1** | **Recruit Pro（採用特化）** | ①② | 中小企業の最大課題は人材。採用サイトは企業サイトと別ロジックが必要で、現行Standardでは対応しきれない。需要が最も高い |
| **2** | **Visual Impact（ビジュアル重視）** | ①③ | 製造業・建設業クライアントが多いなら必須。写真があれば説得力が出る業種向け。Standardとの差別化も明確 |
| **3** | **Lead Gen（リード獲得特化）** | ①② | 「問い合わせを増やしたい」は全クライアント共通の要望。CVR重視の設計は独立テンプレートとして価値あり |
| **4** | **Trust Builder（信頼構築型）** | ②③ | 新規・知名度低い企業向け。社会的証明を効かせる構成は心理学的にも効果実証済み |
| **5** | **Local Business（地域密着）** | ①③ | 建設業・製造業との相性◎。MEO対策を組み込んだテンプレートは競合と差別化できる |
| **6** | **Minimal Pro（洗練ミニマル）** | ③ | 高単価クライアント向け。AIとの差別化を明確にできるデザインアプローチ |
| **7** | **Story（ナラティブ型）** | ② | 理念重視の企業に刺さる。採用にも効く。ただし素材（写真・テキスト）の質に依存 |
| **8** | **Dynamic Flow（ダイナミック）** | ③ | 差別化としては強力だが、使いどころを選ぶ。IT系クライアントが増えたら検討 |

## 実装推奨ロードマップ

**Phase 1（最優先）**
- Recruit Pro → 採用課題を抱えるクライアント多数、即効性あり
- Visual Impact → 製造・建設業向け、写真素材さえあれば展開容易

**Phase 2（次点）**
- Lead Gen → 汎用性高い、既存Standardの派生として効率的に開発可能
- Trust Builder → 新規クライアントに最初に提案しやすい

**Phase 3（拡充）**
- Local Business → 地域密着ビジネス専用として独自価値
- Minimal Pro → 高単価帯クライアントの選択肢として

この順序で実装すれば、クライアントの95%以上のニーズをカバーできます。
