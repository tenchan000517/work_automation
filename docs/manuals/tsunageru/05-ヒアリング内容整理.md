# ヒアリング内容整理マニュアル

## 基本情報

| 項目 | 内容 |
|------|------|
| 商材 | ツナゲル |
| 業務内容（大項目） | 採用求人原稿作成 |
| 担当者 | 河合, 中尾文香 |
| 使用ツール | スプレッドシート |
| 成果物 | 構成案 |
| チェックポイント | 要件漏れ |

## 目的

企業との打ち合わせで得た情報を整理し、求人原稿作成・動画制作のための構成案を作成する。

## シミュレーション（業務の流れ）

**前工程からのトリガー：**
議事録のワークスグループ共有を受けて開始

**業務内容：**
1. NOTTAまたは録音データから議事録を確認
2. ヒアリングシートへ転記
3. 構成案の作成

**トリガー（次工程へ）：**
→ 構成案をワークスで共有

## 作業手順

### Step 1: 打ち合わせ議事録の確認

1. NOTTAまたは録音データから議事録を確認
2. 企業の基本情報を抽出
3. 採用に関する要望・課題を特定

### Step 2: ヒアリングシートへの転記

1. スプレッドシートを開く
2. 以下の項目を記入：
   - 企業基本情報
   - 募集職種
   - 採用要件
   - 企業の強み・特徴
   - 働く環境の情報
   - 採用における課題

### Step 3: 構成案の作成

**ライティング用構成案：**
- 求人原稿の方向性
- 訴求ポイント
- 差別化要素

**動画用構成案：**
- インタビュー質問の方向性
- 伝えたいメッセージ
- 撮影候補場所

### Step 4: 要件漏れチェック

- [ ] 企業の基本情報が揃っているか
- [ ] 募集職種と人数は明確か
- [ ] 採用要件は具体的か
- [ ] 企業の強み・特徴は把握できているか
- [ ] 競合他社との差別化ポイントはあるか
- [ ] 動画で伝えたいメッセージは明確か

## 成果物

- 構成案（スプレッドシート）

## 後工程への引継ぎ

- 原稿執筆担当（河合, 中尾文香）へ構成案を共有
- 動画企画担当（川崎）へ質問設計の方向性を共有
