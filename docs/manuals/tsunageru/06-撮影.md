# 撮影マニュアル（縦動画・ショート動画運用向け）

## 1. マニュアルの目的

本マニュアルは、**編集しやすく、ショート動画として成立する映像素材を、誰でも安定して撮影できる状態を作ること**を目的とする。

## 2. 撮影の大前提（最初に必ず意識すること）

- **縦動画（9:16）で撮影する**
- **HD・60FPSで撮影する**
  - こだわりたい撮影は 4K・60FPS でも可
  - 指示がない限り 4K は使わない
- **寄りすぎない**
  - 編集で「寄る」ことはできる
  - 編集で「引く」ことはできない
- **動画全体の流れ（導入〜オチ）を意識して撮る**

## 3. 画角の基本（人・物 共通）

### 3-1. 画角の考え方

- 真正面からは避ける
- 少し斜め（約20〜30度）から撮る
- 被写体の周囲に余白を残す

### 3-2. 1つのものを撮る場合

- 主役が明確に分かる距離を取る
- 被写体全体＋少し余白が入る構図にする

### 3-3. 複数のものを撮る場合

- すべての被写体が自然に画面内に入る距離を取る
- 手前・奥、左右などで軽く奥行きを作る

## 4. カメラの動かし方

### 4-1. 動かす速度

常に**気持ちゆっくりめ**

**理由：**
- 早くする編集は簡単
- 遅くすると映像に違和感が出やすい

### 4-2. カメラ操作の基本

- 手首だけで振らない
- 一定速度を意識する
- 無意味な動きはしない

## 5. 寄り（アップ）撮影時のルール

**必須：2パターン撮影**

**① 定点撮影**
- カメラ固定
- 5秒以上

**② 流し撮り**
- 左 → 右 または 右 → 左
- ゆっくり一定速度

※どちらか一方だけにならないよう注意

## 6. 引き（ワイド）撮影時の注意点

**関係のないものを映さない**

特に注意するもの：
- 通行人
- 他のお客さん
- 車、ナンバープレート

人が映ると：
- モザイク処理が必要
- 編集工数が増える

**原則：最初から映さない**

## 7. 撮影の基本構成（ジャンル別）

### 7-1. 飲食店（店内飲食）

1. 移動シーン（徒歩・車）
2. 店舗を指さすカット（「ここ！」が分かる）
3. 入店
4. メニュー選び
5. 料理到着
6. 物撮り（寄り・引き）
7. いただきます
8. 食べ進め
9. 途中感想
10. 完食
11. 動画のオチ（👍・表情・一言）

### 7-2. 語りメインの場合

1. 導入（場所・雰囲気）
2. 話し始め（定点）
3. 補足カット（物・周囲）
4. まとめ
5. オチ

### 7-3. テイクアウトの場合

1. 店舗外観
2. 注文
3. 受け取り
4. 開封
5. 物撮り（寄り・引き）
6. 感想
7. オチ

### 7-4. バラエティ系ショート動画の場合

（企画・検証・リアクション系）

**目的：** テンポ・リアクション・編集展開を作りやすい素材を撮ること

1. **導入（フック）** - 一言で内容が分かる、驚き／疑問／ツッコミ
2. **企画説明（短く）** - 1〜2カットで完結
3. **実行シーン** - 行動・検証・チャレンジ、同じ動作は最低2カット以上
4. **リアクション** - 表情アップ、一言コメント、驚き／笑い／困惑
5. **補足カット** - 結果物、周囲の様子、ビフォー／アフター
6. **まとめ or 一言**
7. **オチ** - 表情 / ツッコミ / 無言リアクション / ジェスチャー

**注意点：**
- 迷ったら回し続ける
- 噛んでも止めない
- 同じリアクションを2回撮る
- カメラ目線／外し目線を混ぜる

## 8. 音声に関する考え方

**気にしなくてよいこと：**
- 環境音（※サイレンなど大きな音が入った場合は撮り直す）
- 自分の声が少し入ること

**注意すること：**
- 発言が被らない
  - 同時に喋らない
  - 被った場合は間を空けて撮り直す

## 9. ショート動画向け・最低限の撮影ルール（補足）

- 素材不足に気をつける（カットはできるが追加はできない）
- 同じ構図を長く撮りすぎない
- ズームは基本使わない（編集で対応）
- 「使える素材を多く撮る」意識を持つ

## 10. 撮影後チェック（現場用）

- [ ] 寄りすぎていないか
- [ ] カメラをゆっくり動かせているか
- [ ] 定点＋流しの両方を撮れているか
- [ ] 不要な人・物が映っていないか
- [ ] 導入とオチが撮れているか

## 11. まとめ

**撮影は「編集を楽にするための工程」**

迷ったら：
- 寄らない・ゆっくり・余白を残す
- 動画の流れを常に意識する
