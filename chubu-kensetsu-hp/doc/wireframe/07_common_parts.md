# 共通パーツ設計

## ヘッダー

### 設計意図

**マーケティング視点**:
主要ページへの導線を確保しつつ、CTAボタン（お問い合わせ・電話）を常時表示してCV機会を最大化。

**デザイン視点**:
シンプルで機能的。ロゴ・ナビゲーション・CTAの3要素をバランスよく配置。スクロール時は背景を追加して視認性を確保。

### レイアウト

- 構成: 全幅、fixed（スクロール追従）
- 高さ: 80px（SP: 60px）
- 背景: 透明（スクロール後: #ffffff + 影）
- z-index: 1000
- 内部構成: ロゴ（左）| ナビゲーション（中央）| CTA（右）

### コンテンツ

**ロゴ**:
- 位置: 左端、padding-left 40px（SP: 20px）
- サイズ: 高さ 40px（SP: 32px）
- リンク: /（トップページ）

**ナビゲーション（PC）**:
- 位置: 中央
- 項目:
  - 会社概要（/about）
  - 事業内容（/service）
  - 採用情報（/recruit）
  - お知らせ（/news）
- スタイル:
  - フォントサイズ: 15px
  - フォントウェイト: Medium (500)
  - 色: #333333（透明背景時は#ffffff）
  - letter-spacing: 0.05em
  - 項目間: 40px
  - ホバー: 色 #e67e22

**CTAエリア（PC）**:
- 位置: 右端、padding-right 40px
- 電話番号: 「052-251-3680」
  - フォントサイズ: 16px
  - フォントウェイト: Bold (700)
  - 色: #1a3a5c
  - アイコン: 電話（左に配置）
- お問い合わせボタン:
  - 文言: 「お問い合わせ」
  - スタイル: Primary Small
  - margin-left: 24px

**ハンバーガーメニュー（SP）**:
- 位置: 右端、padding-right 20px
- アイコン: 3本線、24px、色 #333333
- タップ時: フルスクリーンメニュー展開

**SPメニュー展開時**:
- 背景: #1a3a5c
- 高さ: 100vh
- ナビ項目:
  - フォントサイズ: 24px
  - 色: #ffffff
  - 項目間: 32px
  - 中央配置
- 閉じるボタン: 右上、×アイコン、白

### スクロール後の変化

- トリガー: 100pxスクロール後
- 変化:
  - 背景: #ffffff
  - box-shadow: 0 2px 8px rgba(0,0,0,0.08)
  - ナビ・ロゴの色: 通常カラーに変更
- トランジション: 0.3s ease
