# 撮影指示書

## 撮影方針

### 全体のトーン

- **雰囲気**: 人の温かさが伝わる、職人の笑顔、現場の活気、アットホーム
- **避けるべき**: 古臭い、激安・安売り感、いかにも建設会社という堅すぎるイメージ
- **カラートーン**: 自然な色味、明るめ、暖色寄り

### 撮影時の注意

- フィルター加工は行わない（自然な色味を維持）
- コントラストはやや高め（+10%程度）
- 彩度は自然（過度な加工はNG）
- 明るさは明るめ（暗い印象を避ける）

## 撮影時のNG例

### 人物写真でのNG

| NG | OK |
|-----|-----|
| スーツでかしこまったポートレート | 現場で作業着を着た自然な笑顔 |
| スタジオで撮影した証明写真風 | 社員食堂で同僚と談笑する姿 |
| 腕組みして立つポーズ | 図面を指差しながら説明する姿 |
| 真顔・無表情 | 自然な笑顔、話している瞬間 |
| 一人だけの孤立した構図 | 複数人での自然なやり取り |

### 建物写真でのNG

| NG | OK |
|-----|-----|
| 曇天・暗い天気での撮影 | 晴天・明るい時間帯での撮影 |
| 正面からの平面的な構図 | 斜め45度からの立体的な構図 |
| 人がいない寂しい印象 | 人の気配・活気がある |
| 古い・汚れが目立つ | 清潔感・新しさが伝わる |
