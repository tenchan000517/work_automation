# 週間データ集計マニュアル

## 基本情報

| 項目 | 内容 |
|------|------|
| 商材 | ツナゲル |
| 業務内容（大項目） | FB資料作成 |
| 担当者 | 河合 |
| 使用ツール | スプレッドシート・Canva |
| 成果物 | スプレッドシート集計データ |
| チェックポイント | 数値推移 |

## 目的

エンゲージ/Pairsonaの週間データを集計し、企業に対して求人原稿のパフォーマンスを定期的に共有する。

## シミュレーション（業務の流れ）

1. エンゲージログインして週間データの集計（閲覧数・イイね数・応募数）
2. 集計データの共有を各企業ワークスグループに共有

## 前工程

- 原稿公開済み
- 運用開始済み

## 実施タイミング

毎週月曜日（前週分を集計）

## 作業手順

### Step 1: エンゲージにログイン

1. エンゲージ管理画面にアクセス
2. 該当企業のダッシュボードを開く
3. 期間を前週に設定

### Step 2: データ収集

| 指標 | 説明 |
|------|------|
| 閲覧数 | 求人ページが閲覧された回数 |
| イイね数 | 求人に「イイね」がついた数 |
| 応募数 | 応募があった件数 |

### Step 3: スプレッドシートに記入

1. 集計用スプレッドシートを開く
2. 該当週のデータを入力
3. 前週との増減を計算
4. 推移グラフを更新（自動の場合は確認のみ）

### Step 4: 数値推移チェック

- [ ] 前週と比較して大きな変動はないか
- [ ] 異常値がないか確認
- [ ] 応募があった場合は詳細を確認

### Step 5: 企業への共有

1. 週間集計データ共有テンプレートを使用
2. 各企業のワークスグループに共有
3. 特記事項があれば記載

## 集計データ項目

| 項目 | 今週 | 先週 | 増減 |
|------|------|------|------|
| 閲覧数 | - | - | - |
| イイね数 | - | - | - |
| 応募数 | - | - | - |

## 成果物

- スプレッドシート集計データ
- 企業への共有メッセージ

## 後工程への引継ぎ

- 月次FB資料作成時に週間データを活用

## 注意事項

- 数値は正確に記録する
- 前週との比較で異常値があれば原因を調査
- 応募があった場合は応募対応担当と情報共有
