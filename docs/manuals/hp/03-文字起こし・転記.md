# 文字起こし・転記マニュアル

## 基本情報

| 項目 | 内容 |
|------|------|
| 商材 | HP制作 |
| タスク番号 | No.3 |
| 担当者 | 河合 |
| 使用ツール | NOTTA, スプレッドシート（ヒアリングシートのメニュー） |
| 成果物 | Part②ヒアリング情報（39項目） |
| 前工程 | No.2 初回打ち合わせ |
| 後工程 | No.4 JSON出力・原稿生成 |

## 目的

初回打ち合わせの文字起こしから、HP制作に必要なヒアリング情報（39項目）を抽出し、ヒアリングシートのPart②に転記する。

## フロー概要

```
NOTTA文字起こし確認
    ↓
📋 文字起こしを整理（プロンプト生成）
    ↓ ※プロンプトをコピーしてAIに貼り付け
AIがJSON形式で出力
    ↓
📥 AI出力を転記
    ↓
Part②に39項目が転記される
    ↓
内容確認・修正
    ↓
ステータス更新
```

## 作業手順

### Step 1: NOTTA文字起こしの確認

1. **NOTTAにログイン**してプロジェクト一覧を開く
2. 該当の打ち合わせの**文字起こしを表示**
3. 内容を確認し、**必要に応じて修正**
   - 固有名詞（会社名、人名など）の誤変換を修正
   - 明らかな誤認識を修正
4. 文字起こし全文を**コピー**

### Step 2: プロンプト生成（GAS使用）

1. **ヒアリングシート**を開く
2. メニュー「**2.📝 ヒアリング反映**」→「**📋 文字起こしを整理（プロンプト生成）**」をクリック

   ![ヒアリング反映メニュー](/images/hp/menu-hearing-hanei.png)

3. ダイアログが表示される：
   - **対象企業を選択**（ドロップダウン）
   - **文字起こしを貼り付け**（テキストエリア）

4. **対象企業を選択**
   - 企業シートを選択すると、企業名がプロンプトに自動挿入される

5. **文字起こしを貼り付け**
   - NOTTAからコピーした文字起こしを貼り付け

6. **「💾 シートに保存」**（任意）
   - 文字起こし原文をPart④に保存しておくと、後で参照可能

7. **「完成版プロンプト」の「コピー」ボタン**をクリック
   - テンプレート + 文字起こしが結合されたプロンプトがコピーされる

### Step 3: AIでヒアリング情報を抽出

1. **Claude / ChatGPT** などのAIツールを開く
2. コピーしたプロンプトを**貼り付けて送信**
3. AIが**JSON形式**でヒアリング情報を出力
   - 39項目が6カテゴリに整理される
4. AIの出力（JSON）を**コピー**

**AIの出力例：**
```json
{
  "企業名": "株式会社○○",
  "ゴール・コンバージョン": {
    "メインのコンバージョン": "問い合わせフォームからの相談",
    "ハードル設定": "無料相談を設ける"
  },
  "ターゲット": {
    "年齢層・性別": "30〜50代男性",
    ...
  },
  ...
}
```

### Step 4: AI出力を転記（GAS使用）

1. メニュー「**2.📝 ヒアリング反映**」→「**📥 AI出力を転記**」をクリック

   ![ヒアリング反映メニュー](/images/hp/menu-hearing-hanei.png)

2. ダイアログが表示される：

   ![AI出力転記ダイアログ - JSON貼り付け](/images/hp/dialog-ai-output-json.png)

   - **転記先企業を選択**（ドロップダウン）
   - **AIが出力したJSONを貼り付け**

3. **転記先企業を選択**
   - Step 2で選択した企業と同じシートを選択

4. **JSONを貼り付け**
   - AIが出力したJSONをそのまま貼り付け
   - ※コードブロック（```json）で囲まれていても自動除去

5. **「💾 シートに保存」**（任意）
   - JSONをPart④に保存しておくと、後で再転記可能

6. **「🔍 解析して比較」**ボタンをクリック
   - JSONが解析され、転記内容の確認画面に遷移

### Step 5: 転記内容の確認・実行

1. **比較画面**が表示される：

   ![AI出力転記ダイアログ - 比較画面](/images/hp/dialog-ai-output-compare.png)

   - 項目名
   - 現在の値（既にPart②に入力済みの値）
   - 新しい値（JSONから抽出された値）

2. **チェックボックスで転記する項目を選択**
   - デフォルトでは新しい値がある項目すべてにチェック
   - 上書きしたくない項目はチェックを外す

3. **「新しい値」は編集可能**
   - 必要に応じて値を修正してから転記

4. **「✅ チェック項目を転記」**ボタンをクリック
   - 選択した項目がPart②に転記される

5. 転記完了のメッセージが表示されたら**ダイアログを閉じる**

### Step 6: 内容確認・補足入力

1. **ヒアリングシートのPart②を確認**
   - 転記された内容が正しいか
   - 抜け漏れがないか

2. **手動で補足・修正**
   - AIが抽出できなかった情報を手動入力
   - 誤った情報を修正

### Step 7: ステータス更新

1. メニュー「**2.📝 ヒアリング反映**」→「**✏️ ステータス更新**」をクリック
2. 以下を更新：
   - **現在タスク**: 4.JSON出力・原稿生成
   - **タスク保持者**: 河合（継続）
   - **状態**: 対応中
3. **「更新」**ボタンをクリック

## トラブルシューティング

### JSONの解析に失敗する場合
- AIの出力がJSON形式になっているか確認
- 余計なテキスト（「以下のJSONです」など）が含まれていないか確認
- コードブロックの記法が崩れていないか確認

### 企業名が一致しない警告が出る場合
- 選択したシートとJSONの企業名が異なると警告が出る
- 正しいシートを選択し直すか、問題なければそのまま続行

### マッピングを確認したい場合
- メニュー「2.📝 ヒアリング反映」→「🔧 マッピング確認」
- 各項目がPart②のどの行に転記されるか確認できる
