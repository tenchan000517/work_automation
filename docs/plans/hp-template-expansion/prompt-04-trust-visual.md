## テンプレート概要

**テンプレート名**: Trust Visual

**型（Type）**: Trust Builder（信頼構築）
**表現（Design）**: Visual Impact（ビジュアル重視）

**コンセプト**:
知名度の低い企業が、**写真と数字で信頼を勝ち取る**。
「みんなが選んでいる」「専門家が認めている」という**社会的証明を全面展開**。
初訪問者の「本当に大丈夫？」という不安を、ビジュアルで払拭する。

**想定クライアント**:
- 新規参入企業、スタートアップ
- 知名度がまだ低い企業
- 新規事業を始めた企業
- 「うちは良い仕事をしているのに伝わらない」と感じている企業

**想定ユーザー（閲覧者）**:
- 初めてこの会社を知った人: 「聞いたことないけど大丈夫？」
- 比較検討中のユーザー: 「他社とどう違うの？」
- 紹介で来た人: 「紹介されたけど、本当に信頼できる？」

**このテンプレートで達成すべきこと**:
1. 初訪問者の離脱率低減
2. 「この会社は信頼できる」という印象形成
3. 問い合わせへのハードル低減
4. 実績・数字による説得力の向上

---

## 技術的制約（これらは変更不可）

1. **ベース**: template-fullorder からクローン
2. **データ管理**: site.json + 各ページ上部のコンテンツデータ定数
3. **スタイル**: Tailwind CSS 4.x（globals.css の @theme 内ブランドカラーのみ変更可）
4. **アニメーション**: Framer Motion 12.x
5. **画像**: next/image コンポーネント使用
6. **レスポンシブ**: PC優先、SP対応必須

## 禁止事項

1. **汎用的な表現の禁止**
   - ❌「数字を表示」
   - ✅「画面中央に4つの数字を横並び。各数字は72px太字、単位は24px。スクロールで画面に入った時にカウントアップアニメーション」

2. **理由なき設計の禁止**
   - ❌「お客様の声を配置」
   - ✅「ファーストビュー直下にお客様の声を配置。理由: 社会的証明は早い段階で提示するほど効果が高いため」

3. **AIデザインの典型パターン禁止**
   - ❌ 全要素にカードコンテナ
   - ❌ 紫系グラデーション
   - ❌ 全セクションで同じ余白
   - ❌ 全要素にフェードインアニメーション

4. **曖昧な指示の禁止**
   - ❌「適切な余白を設ける」
   - ✅「上余白60px、下余白80px。下を大きくする理由: 次のセクションへの期待感を持たせるため」
