## フロー概要

```
【HP実装完了】Claude CodeがHANDOFFに必要素材リストJSONを出力
    ↓
【素材フォルダ作成】GASメニュー「📂 素材フォルダ作成（JSON入力）」
├── Googleドライブにフォルダ構造が作成される
│   ├── 企業名/撮影素材/（ページごとにサブフォルダ）
│   └── 企業名/本番素材/（destPath構造）
└── 撮影指示書スプレッドシートが自動生成される
    ↓
【撮影指示書を共有】撮影担当に編集権限で共有
    ↓
【現地撮影】撮影指示書を参照しながら撮影
    ↓
【データ納品】撮影素材フォルダにアップロード
    ↓
【素材選定】撮影素材から選定 → 本番素材フォルダに配置
    ↓
【素材反映】GASメニュー「📷 素材反映」でClaude Code指示生成
    ↓
【素材チェック】GASメニュー「✅ 素材チェック実行」で確認
    ↓
No.7 素材反映・チェックへ
```

## GASメニュー: 素材フォルダ作成

HP実装完了後、HANDOFFに出力された必要素材リストJSONを使って素材フォルダを作成します。

### 方法1: JSON入力（推奨）

1. ヒアリングシートを開く
2. メニュー「**3.📁 素材フォルダ**」→「**📂 素材フォルダ作成（JSON入力）**」をクリック
3. ダイアログが表示される
4. **HANDOFFの必要素材リストJSONを貼り付け**
   - Claude Codeが出力したJSONをそのままコピー＆ペースト
5. **「フォルダ作成」**ボタンをクリック
6. 以下が自動作成される:
   - **Googleドライブ: フォルダ構造**
     - `企業名/撮影素材/`（ページごとにサブフォルダ）
     - `企業名/本番素材/`（destPath構造）
   - **撮影指示書スプレッドシート**
     - ガイドシート: フロー説明、フォルダリンク
     - 素材リストシート: 撮影指示詳細（✅チェック欄）
7. Part④に各URLが自動保存される

### 方法2: 企業選択（従来方式）

1. ヒアリングシートを開く
2. メニュー「**3.📁 素材フォルダ**」→「**🆕 素材フォルダ作成（企業選択）**」をクリック
3. 対象企業を選択
4. 必要なページをチェック（TOP, About, Service等）
5. 「フォルダ作成」ボタンをクリック

## 撮影指示書スプレッドシート

「📂 素材フォルダ作成（JSON入力）」で自動生成されるスプレッドシートです。

### シート構成

| シート名 | 内容 |
|----------|------|
| **ガイド** | 素材管理フローの説明、フォルダリンク一覧 |
| **素材リスト** | 撮影対象一覧、撮影指示、チェック欄 |

### 素材リストシートの列

| 列 | 内容 |
|----|------|
| ID | 素材識別子（hero-top等） |
| ファイル名 | 保存するファイル名 |
| ページ | 使用ページ（TOP, About等） |
| 用途 | 使用箇所の説明 |
| 撮影指示 | 撮影時の注意点 |
| 推奨サイズ | 画像サイズ（1920x1080px等） |
| ✅撮影完了 | 撮影担当がチェック |
| ✅配置完了 | 素材チェック実行で自動更新 |
| 本番URL | 素材チェック実行で自動転記 |

### 使い方

1. 撮影指示書URLを撮影担当に共有（編集権限）
2. 撮影担当は「素材リスト」シートを見ながら撮影
3. 撮影完了したら「✅撮影完了」欄にチェック
4. 撮影素材フォルダにアップロード

## 1. マニュアルの目的
本マニュアルは、HP制作に必要な写真・動画素材を、
誰でも安定した品質で撮影できる状態を作ることを目的とする。

## 2. 撮影の大前提（最初に必ず意識すること）

### 静止画（写真）撮影
- 高解像度で撮影する（最低でも1920×1080以上）
- RAW撮影が可能な場合はRAWで撮影

### 動画撮影
- 縦動画（9:16）または横動画（16:9）※用途に応じて選択
- HD・60FPSで撮影する
  - こだわりたい撮影は 4K・60FPS でも可
  - 指示がない限り 4K は使わない

### 共通
- 寄りすぎない
  - 編集で「寄る」ことはできる
  - 編集で「引く」ことはできない
- 被写体の周囲に余白を残す

## 3. 画角の基本（人・物 共通）

### 3-1. 画角の考え方
- 真正面からは避ける
- 少し斜め（約20〜30度）から撮る
- 被写体の周囲に余白を残す

### 3-2. 1つのものを撮る場合
- 主役が明確に分かる距離を取る
- 被写体全体＋少し余白が入る構図にする

### 3-3. 複数のものを撮る場合
- すべての被写体が自然に画面内に入る距離を取る
- 手前・奥、左右などで軽く奥行きを作る

## 4. HP素材撮影の基本構成

### 必須素材
- 外観写真（全景・入口・看板）
- 内観写真（複数アングル）
- スタッフ写真（個人・集合）
- 業務風景（作業中の様子）
- 設備・機材（必要に応じて）
- 商品・サービス（該当する場合）

### 撮影時の注意
- 人物撮影時は表情が自然になるよう声かけする
- 照明の当たり方を確認（逆光を避ける）
- 背景に余計なものが映らないよう整理する

## 5. 動画撮影時のカメラの動かし方

### 5-1. 動かす速度
常に気持ちゆっくりめ

理由：
- 早くする編集は簡単
- 遅くすると映像に違和感が出やすい

### 5-2. カメラ操作の基本
- 手首だけで振らない
- 一定速度を意識する
- 無意味な動きはしない

## 6. 引き（ワイド）撮影時の注意点
関係のないものを映さない

特に注意するもの：
- 通行人
- 他のお客さん
- 車、ナンバープレート

人が映ると：
- モザイク処理が必要
- 編集工数が増える

原則：最初から映さない

## 7. 音声に関する考え方（動画撮影時）

### 気にしなくてよいこと
- 環境音（※サイレンなど大きな音が入った場合は撮り直す）
- 自分の声が少し入ること

### 注意すること
- 発言が被らない
  - 同時に喋らない
  - 被った場合は間を空けて撮り直す

## 8. 撮影後チェック（現場用）
- [ ] 必須素材がすべて揃っているか
- [ ] 寄りすぎていないか
- [ ] カメラをゆっくり動かせているか（動画の場合）
- [ ] 不要な人・物が映っていないか
- [ ] ピントが合っているか
- [ ] 露出は適切か（明るすぎ・暗すぎではないか）

## 9. データ納品ルール
1. 共有ドライブの指定フォルダを開く
2. 指定のファイル名で格納

### ファイル命名規則
`[企業名]_[撮影日]_[種別]_[連番].jpg / .mp4`

例:
- 中京商運_20251226_外観_01.jpg
- 中京商運_20251226_動画_01.mp4

## 10. まとめ
撮影は「デザイン・実装を楽にするための工程」

迷ったら：
- 寄らない・ゆっくり・余白を残す
- 必要な素材が揃っているか常に意識する
