# 動画編集マニュアル

## 基本情報

| 項目 | 内容 |
|------|------|
| 商材 | ツナゲル |
| 業務内容（大項目） | インタビューショート動画 |
| 担当者 | 河合 |
| 使用ツール | Powerdirector |
| 成果物 | ショート動画 |
| チェックポイント | テロップ誤字 |

## 目的

撮影された動画素材を編集し、採用訴求に効果的なショート動画を完成させる。

## 前工程

- 撮影完了
- 撮影データが共有ドライブに格納済み

## 準備

### データ取得

1. 共有ドライブから撮影データをローカルにダウンロード
2. 元データをドライブに格納（バックアップ）

## 作業手順

### Step 1: 素材の確認

1. 撮影データをすべて確認
2. 使用する部分をピックアップ
3. 音声品質を確認

### Step 2: 編集作業

1. Powerdirectorでプロジェクト作成
2. 縦型（9:16）で設定
3. 素材を読み込み
4. カット編集
   - 不要な部分をカット
   - 間延びした部分を調整
   - ジャンプカットで飽きさせない構成に

### Step 3: テロップ入れ

1. 発言内容をテキスト化
2. テロップのスタイルを統一
3. 読みやすいフォント・サイズを選択
4. 表示タイミングを調整

### Step 4: BGM・SE

1. 適切なBGMを選択
2. 音量バランスを調整（声が聞こえることが最優先）
3. 必要に応じてSE（効果音）を追加

### Step 5: 書き出し

1. ショート動画用の設定で書き出し
2. 推奨設定：
   - 解像度: 1080x1920（縦型）
   - フレームレート: 30fps
   - フォーマット: MP4

## チェックリスト（書き出し前）

### テロップ誤字チェック（最重要）

- [ ] 人名の表記は正しいか
- [ ] 会社名の表記は正しいか
- [ ] 専門用語のスペルは正しいか
- [ ] 誤変換はないか

### その他チェック

- [ ] 音声と映像の同期はずれていないか
- [ ] BGMの音量は適切か
- [ ] テロップの表示タイミングは適切か
- [ ] 動画の長さはショート動画として適切か（60秒以内推奨）
- [ ] 冒頭3秒で視聴者の興味を引く構成か

## 成果物

- ショート動画（MP4ファイル）

## データ格納

**ファイル命名規則：**
`[企業名]_[撮影日]_edit_v[バージョン].mp4`

例: `中京商運_20251226_edit_v1.mp4`

**格納場所：**
指定のドライブに指定ファイル名で格納

## 後工程への引継ぎ

- 原稿執筆担当へ動画完成の連絡
- 動画を原稿に挿入する準備
