## 2. 新旧バージョンの違い

### 新バージョンの特徴

1. **画像にキャプション付き**
```typescript
images: [
  { url: "/images/xxx.png", caption: "STEP X: 説明" }
]
```

2. **hasInputField付きpopup（入力フィールド付きテンプレート）**
```typescript
links: [{
  label: "議事録作成プロンプト",
  type: "popup",
  hasInputField: true,
  inputSectionTitle: "...",
  inputLabel: "...",
  template: `...{{input}}...`
}]
```

3. **GASダイアログ連携**
- 設定シートから担当者名を動的読み込み
- コピペボタンで効率的にコピー

### 旧バージョンの特徴

1. **画像がキャプションなし（単純文字列）**
```typescript
images: ["/images/xxx.png"]
```

2. **content直接埋め込みpopup**
```typescript
links: [{ label: "フォーマット", type: "popup", content: `...` }]
```

3. **descriptionに例文・担当者名がハードコード**
```typescript
description: `...
@河合 cc:@青柳
...`
```
