# Newsページ 詳細設計

## ページ概要

| 項目 | 内容 |
|-----|------|
| 目的 | 会社の最新情報を発信し、サイトの更新感と活動実態を伝える |
| ターゲット | 既存顧客、取引先、求職者、検索エンジン（SEO） |
| CV導線上の役割 | 興味維持（リピート訪問促進、SEO効果） |
| 滞在時間目標 | 1分 |
| 次のアクション | 各記事の詳細閲覧、または他ページへの回遊 |

---

## セクション1: ページヘッダー

Aboutページと同様のスタイル

**コンテンツ**:
- ページタイトル: 「お知らせ」
- 英語表記: 「NEWS」
- パンくず: TOP > お知らせ

## セクション2: カテゴリフィルター

### 設計意図

**マーケティング視点**:
ユーザーが目的の情報に素早くアクセスできるようカテゴリ別フィルターを設置。

### レイアウト

- 構成: コンテナ幅（max-width: 900px）、横並びタブ
- 高さ: auto
- 背景: #ffffff
- 余白: 上60px 下40px 左右80px（SP: 上40px 下24px 左右20px）

### コンテンツ

**カテゴリタブ**:
- すべて（デフォルト選択）
- お知らせ
- 採用情報
- 施工実績

### タブスタイル

- 非選択時:
  - 背景: transparent
  - 色: #666666
  - padding: 12px 24px
  - border-bottom: 2px solid transparent
- 選択時:
  - 色: #1a3a5c
  - border-bottom: 2px solid #1a3a5c
- ホバー: 色 #1a3a5c

---

## セクション3: お知らせ一覧

### 設計意図

**デザイン視点**:
シンプルなリスト形式で情報を整理。日付・カテゴリ・タイトルを明確に区分し、読みやすさを優先。

### レイアウト

- 構成: コンテナ幅（max-width: 900px）
- 高さ: auto
- 背景: #ffffff
- 余白: 上0 下120px 左右80px（SP: 上0 下80px 左右20px）

### コンテンツ

**お知らせリスト**:

| 日付 | カテゴリ | タイトル |
|-----|---------|---------|
| 2026.01.15 | お知らせ | ホームページをリニューアルしました |
| 2026.01.10 | 採用情報 | 2026年度新卒採用エントリー受付開始 |
| 2025.12.20 | 施工実績 | 愛知県立工業高校体育館改修工事が竣工しました |
| 2025.12.01 | お知らせ | 年末年始休業のお知らせ（12/29〜1/3） |
| 2025.11.15 | 採用情報 | 会社説明会を開催します（12月15日） |
| 2025.11.01 | 施工実績 | 名古屋市内マンション新築工事が竣工しました |
| 2025.10.20 | お知らせ | 社内研修のため10/25は電話対応を休止します |
| 2025.10.01 | 採用情報 | 中途採用（施工管理）の募集を開始しました |

### リストアイテムスタイル

- 構成: 横並び（日付 | カテゴリ | タイトル）（SP: 縦並び）
- 区切り線: border-bottom: 1px solid #e8e6e3
- padding: 24px 0
- ホバー: 背景 #f8f7f4

- 日付:
  - 幅: 120px（SP: 100%）
  - フォントサイズ: 14px
  - 色: #666666
- カテゴリラベル:
  - 幅: 100px（SP: inline）
  - フォントサイズ: 12px
  - 色: #1a3a5c
  - 背景: #e8e6e3
  - padding: 4px 12px
  - 角丸: 2px
- タイトル:
  - flex: 1
  - フォントサイズ: 16px（SP: 15px）
  - 色: #333333
  - ホバー: 色 #e67e22
  - margin-left: 24px（SP: margin-top 12px）

  - 色: #666666

## セクション5: 記事詳細ページ（テンプレート）

### 設計意図

**デザイン視点**:
記事本文は読みやすさを最優先。適切な余白と行間で長文でも読み疲れないデザイン。

### レイアウト

- 構成: コンテナ幅（max-width: 700px）
- 高さ: auto
- 背景: #ffffff
- 余白: 上80px 下120px 左右80px（SP: 上60px 下80px 左右20px）

### コンテンツ

**記事メタ情報**:
- 日付: 「2026.01.15」
  - フォントサイズ: 14px
  - 色: #666666
- カテゴリ: 「お知らせ」
  - 前述のラベルスタイル

**記事タイトル**:
```
ホームページをリニューアルしました
```
- フォントサイズ: PC 28px / SP 22px
- フォントウェイト: Bold (700)
- 色: #1a3a5c
- margin-top: 16px
- margin-bottom: 40px

**記事本文**:
```
いつも中部建設をご愛顧いただき、誠にありがとうございます。

このたび、より使いやすく、より分かりやすいホームページを目指して、全面リニューアルを行いました。

主な変更点は以下の通りです。

・スマートフォンでの閲覧に最適化しました
・サービス内容をより詳しくご紹介しています
・施工事例を多数追加しました
・採用情報ページを充実させました

今後とも中部建設をよろしくお願いいたします。

ご不明な点がございましたら、お気軽にお問い合わせください。
```

### 本文スタイル

- フォントサイズ: 16px
- 色: #333333
- 行間: 2.0
- 段落間: 28px
- リスト:
  - 左padding: 24px
  - 行間: 1.8
  - マーカー: 「・」色 #e67e22

**前後の記事リンク**:
- 位置: 記事下部
- margin-top: 64px
- border-top: 1px solid #e8e6e3
- padding-top: 32px
- 2カラム（前の記事 / 次の記事）

**一覧へ戻るリンク**:
- 文言: 「お知らせ一覧へ戻る」
- スタイル: Ghost
- margin-top: 40px
- text-align: center
