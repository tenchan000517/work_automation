# 構成案 → 用途別変換プロンプト集

## 概要

AI構成案生成で出力された構成案を、用途別のフォーマットに変換するプロンプト。
2段階方式で使用する。

```
構成案（AI出力）
    ↓ 用途別プロンプトで変換
    ├→ 【A】ペアソナ/エンゲージ入力用
    ├→ 【B】ワークス報告用サマリー
    └→ 【C】撮影指示書
```

---

## 【B】ワークス報告用サマリー変換プロンプト

### 用途
- 構成案の完成をワークスで報告する際の投稿文
- 原稿チーム・動画チームへの共有

### プロンプト

```
以下の構成案を、LINEWORKSで共有するための報告サマリーに変換してください。

【出力ルール】
1. 冒頭に報告の要点（1〜2行）
2. 構成案の概要を箇条書きで簡潔に

4. 次のアクションを明記
5. 全体で400文字以内
```

## 使用フロー

### Step 1: 構成案を生成
```
GAS「📝 構成案生成」→「📋 構成案を生成」
    ↓
AIに貼り付けて実行
    ↓
構成案を取得
```

### Step 2: 用途別に変換
```
【ペアソナ/エンゲージ入力】
構成案 + 変換プロンプトA → AIに貼り付け → コピペ用テキスト

【ワークス報告】
構成案 + 変換プロンプトB → AIに貼り付け → 報告サマリー

【撮影指示書】
構成案 + 変換プロンプトC → AIに貼り付け → 撮影指示書
```
