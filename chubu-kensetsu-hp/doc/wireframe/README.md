## 各ファイルの内容

### 00_overview.md
サイト全体の戦略設計を記載。3人の専門家（WEBマーケティング、求人コンサルタント、WEBデザイナー）の視点から設計。

- マーケティング戦略（コンバージョンファネル、心理トリガー、ABテスト項目）
- 採用ブランディング戦略（求職者インサイト、情報開示戦略、感情設計）
- ブランド体験設計（カラーパレット、タイポグラフィ、余白設計、UIコンポーネント）

### 01_top.md 〜 06_news.md
各ページの詳細設計。セクションごとに以下を記載。

- 設計意図（マーケティング/採用/デザイン視点）
- レイアウト仕様
- コンテンツ（文言、見出し、CTA）
- 画像指示
- アニメーション仕様
- UI詳細

### 07_common_parts.md
サイト共通で使用するパーツの設計。

- ヘッダー
- フッター
- ボタンコンポーネント
- フォームコンポーネント
- その他UIパーツ

### 08_photo_guide.md
撮影が必要な写真の指示書。

- 撮影方針
- 各写真の詳細指示（用途、内容、構図、サイズなど）
- NG例

1. **TOP** - ファーストタッチポイント、ターゲット別導線

5. **Contact** - お問い合わせフォーム、採用エントリーフォーム
6. **News** - お知らせ一覧、記事詳細
