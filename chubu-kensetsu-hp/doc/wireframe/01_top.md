# TOPページ 詳細設計

## ページ概要

| 項目 | 内容 |
|-----|------|
| 目的 | 会社の第一印象を決定づけ、ターゲット別に適切なページへ誘導する |
| ターゲット | 全訪問者（法人顧客、求職者、一般） |
| CV導線上の役割 | 認知→興味（ファーストタッチポイント） |
| 滞在時間目標 | 1分30秒 |
| 次のアクション | 法人→Service/Contact、求職者→Recruit、情報収集→About |

## セクション1: ファーストビュー

### 設計意図

**マーケティング視点**:
3秒で「何の会社か」「信頼できるか」を伝える。創業40年と公共事業実績で権威性を即座に訴求。CTAは2つ設置し、法人顧客と求職者の両方に対応。

**デザイン視点**:
全幅の写真で没入感を演出。人物写真を使用し「人の温かさ」を第一印象で伝える。テキストは左寄せ配置で視線を自然に誘導。

### レイアウト

- 構成: 全幅（100vw）、背景画像+オーバーレイ+左寄せテキスト
- 高さ: 100vh（min-height: 600px）
- 背景: 写真+暗めのオーバーレイ（rgba(26, 58, 92, 0.5)）
- 余白: コンテンツは左から80px（SP: 20px）、上下中央配置
- モバイル時: テキストサイズ縮小、CTAボタンは縦並び

### コンテンツ

**キャッチコピー**:
```
創業40年。名古屋で選ばれ続ける理由がある。
```
- フォントサイズ: PC 48px / SP 28px
- フォントウェイト: Bold (700)
- 色: #ffffff
- 行間: 1.4
- 最大幅: 600px

**サブキャッチコピー**:
```
設計から施工、アフターフォローまで。
地域に根ざした確かな技術で、お客様の想いを形にします。
```
- フォントサイズ: PC 18px / SP 15px
- フォントウェイト: Regular (400)
- 色: rgba(255, 255, 255, 0.9)
- 行間: 1.8
- margin-top: 24px
- 最大幅: 500px

**CTAボタン群**:
- margin-top: 40px
- ボタン間: 16px

**ボタン1（法人向け）**:
- 文言: 「まずは無料相談」
- リンク先: /contact
- スタイル: Primary Large
- サイズ: padding 18px 48px

**ボタン2（求職者向け）**:
- 文言: 「採用情報を見る」
- リンク先: /recruit
- スタイル: Secondary Large（背景透明、白枠、白文字）
- サイズ: padding 18px 48px

### 画像指示

- 内容: 現場で複数の職人が作業している様子。若手とベテランが一緒に図面を確認している場面が望ましい。ヘルメット・作業着着用。自然な笑顔が見える瞬間。
- 撮影: 広角レンズ、斜め上からやや俯瞰気味、自然光（午前中推奨）
- サイズ: 1920px × 1080px以上、16:9比率
- 処理: 明るさ+10%、彩度そのまま

### スクロールインジケーター

- 位置: 画面下部中央、bottom 40px
- 要素: 下向き矢印アイコン（線画、白、2px stroke）
- アニメーション: 上下に8pxバウンス、2秒ループ、ease-in-out

### アニメーション

- 要素: キャッチコピー、サブキャッチ、CTAボタン

- 動き: フェードイン+左から20pxスライド
- 速度: 0.8秒
- イージング: ease-out
- 遅延: キャッチ0秒、サブ0.2秒、CTA0.4秒
