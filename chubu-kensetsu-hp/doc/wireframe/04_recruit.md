# Recruitページ 詳細設計

## ページ概要

| 項目 | 内容 |
|-----|------|
| 目的 | 会社の魅力を伝え、「ここで働きたい」と思わせ、応募につなげる |
| ターゲット | 高校新卒者、中途採用希望者（施工管理経験者） |
| CV導線上の役割 | 認知→興味→比較検討→行動（応募完結まで） |
| 滞在時間目標 | 5分 |
| 次のアクション | エントリーフォーム送信 |

## セクション1: ファーストビュー（採用メイン）

### 設計意図

**採用視点**:
「人」を前面に出し、「この人たちと働きたい」という感情を喚起。数字よりも雰囲気を先に伝える。

**デザイン視点**:
明るく温かみのある写真を大きく使用。テキストは控えめに、写真が語る構成。

### レイアウト

- 構成: 全幅、背景画像+オーバーレイ+中央テキスト
- 高さ: 80vh（min-height: 500px）
- 背景: 写真+オーバーレイ（rgba(26, 58, 92, 0.4)）
- テキスト: 中央配置

### コンテンツ

**キャッチコピー**:
```
技術も、人も、
一緒に育てていく会社です。
```
- フォントサイズ: PC 40px / SP 26px
- フォントウェイト: Bold (700)
- 色: #ffffff
- 行間: 1.5
- text-align: center

**サブキャッチコピー**:
```
創業40年。定着率90%以上。
ベテランから若手へ、確かな技術と温かい心を受け継いでいます。
```
- フォントサイズ: PC 18px / SP 15px
- 色: rgba(255, 255, 255, 0.9)
- 行間: 1.8
- margin-top: 24px
- text-align: center

**CTAボタン**:
- 文言: 「募集要項を見る」
- リンク先: #requirements
- スタイル: Primary Large（白背景、紺文字）
- margin-top: 40px

### 画像指示

- 内容: 社員5-6人が笑顔で集まっている写真。現場または社員食堂で、若手とベテランが混ざっている。自然な談笑シーン。ヘルメット・作業着着用可。
- サイズ: 1920px × 1200px
- 処理: 明るめ、暖色寄り

### アニメーション

- 要素: キャッチコピー
- トリガー: ページ読み込み完了時
- 動き: フェードイン
- 速度: 0.8秒

