# 打ち合わせ前準備マニュアル

## 目的

初回打ち合わせをスムーズに行うための事前準備。
先方のフォーム回答からヒアリングシートを作成し、撮影可能日程も事前に確認しておく。

## 作業手順

### 1. ヒアリングシート作成

1. ヒアリングシートのスプレッドシートを開く
2. メニュー「📋 ヒアリングシート」→「🆕 新規作成（フォーム回答から）」をクリック
3. 該当企業のフォーム回答を選択
4. 「実行」をクリック → 企業名のシートが自動作成される
5. 作成されたシートを開き、データが正しく転記されているか確認

### 2. 撮影可能日程の確認

1. 撮影担当者に連絡し、撮影可能な日程を5候補程度確認
2. 初回打ち合わせで先方に提示できるよう準備
3. ワークスで撮影担当者と候補日を共有

## 成果物

- ヒアリングシート（企業名入り、フォームデータ転記済み）
- 撮影可能日程（5候補程度）

## 備考

- フォーム回答が来ていない場合は「新規作成（手動）」で空のシートを作成
- 撮影日程は初回打ち合わせで先方と最終調整
