## 目的

応募者からの応募に対して迅速かつ適切に対応し、企業の採用成功をサポートする。

## シミュレーション（業務の流れ）

応募対応フロー使用し応募者への連絡・企業への連絡

## 前工程

- キックオフMTG完了
- 原稿公開済み
- 運用開始

## 応募対応フロー

```
応募受信 → 応募者情報の確認 → 応募者へ初回連絡（24時間以内） → 企業へ応募報告 → 面接日程調整 → 面接実施（企業側） → 結果報告
```

## 作業手順

### Step 1: 応募受信の確認

1. エンゲージ/Pairsonaにログイン
2. 新規応募を確認
3. 応募者情報を確認
   - 名前
   - 連絡先
   - 応募動機
   - 職歴等

### Step 2: 応募者への初回連絡

**※24時間以内に連絡することが重要**

1. 電話連絡を試みる
2. 電話がつながらない場合はメールを送信
3. 通話ログを記録

**初回連絡の内容：**
- 応募へのお礼
- 求人内容の確認
- 面接希望日程のヒアリング
- 不明点の確認

### Step 3: 企業への報告

1. ワークスで企業担当者に連絡
2. 応募者情報を共有
3. 面接可能日程を確認

### Step 4: 面接日程調整

1. 応募者と企業の日程をすり合わせ
2. 確定日程を双方に連絡
3. 面接詳細（場所・持ち物等）を応募者に案内

### Step 5: 面接後フォロー

1. 企業から結果を確認
2. 応募者に結果を連絡
3. 記録を更新

## 連絡テンプレート

### 応募者への初回メール

```
件名: 【[企業名]】ご応募ありがとうございます

[応募者名]様

この度は[企業名]の求人にご応募いただき、
誠にありがとうございます。

採用担当の[担当者名]と申します。

ご応募内容を確認させていただきました。
つきましては、面接日程についてご相談させてください。

ご都合の良い日程を2〜3候補いただけますと幸いです。
（平日・土日/午前・午後などご希望があればお知らせください）

ご不明点がございましたら、お気軽にお問い合わせください。

何卒よろしくお願いいたします。

--
署名
```

### 企業への応募報告

```
[企業名]様

お世話になっております。
Singの[担当者名]です。

本日、求人への新規応募がございましたのでご報告いたします。

■応募者情報
お名前:
年齢:
連絡先:
応募動機:
職歴:

■ご対応のお願い
面接可能な日程をお知らせください。
応募者様との日程調整を進めさせていただきます。

何卒よろしくお願いいたします。
```
