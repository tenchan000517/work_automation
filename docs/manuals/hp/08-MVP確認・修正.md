## 目的

先方にMVPを送付し、フィードバックをもらい、修正対応を行い、最終承認を取得する。

## フロー概要

```
【STEP 1】MVP送付
    ↓ URLとトークスクリプトを共有
先方確認
    ↓
【STEP 2】FB収集
    ↓ 打ち合わせ or メールで修正点を確認
修正リスト作成
    ↓
【STEP 3】修正対応
    ↓ Claude Codeで修正
修正版URL共有
    ↓
【STEP 4】再確認・承認
    ↓ 先方確認 → 承認
承認取得
    ↓
ステータス更新
```

## 作業手順

### STEP 1: MVP送付

1. **Vercelにデプロイ**（開発環境のURLを共有する場合）
   ```bash
   vercel
   ```
   または、開発サーバーのURLを共有

2. **メールでMVP送付**

   **メール例:**
   ```
   件名: 【HP制作】MVP確認のお願い（株式会社○○様）

   ○○様

   お世話になっております。
   Singの河合です。

   HPのMVP（最小機能版）が完成いたしましたので、
   ご確認をお願いいたします。

   ■ 確認用URL
   https://[プロジェクト名].vercel.app/

   ■ ご確認いただきたい点
   ・デザインの方向性
   ・掲載内容の正確性
   ・修正したい箇所

   ■ 確認期限
   ○月○日（○）まで

   ご不明点やご質問がございましたら、
   お気軽にお知らせください。

   よろしくお願いいたします。
   ```

3. **オンライン打ち合わせの場合**
   - Google Meetで画面共有
   - トークスクリプトを元にデザインの意図を説明
   - その場でFBを収集

---

### STEP 2: FB収集

1. **FBの整理**
   - 修正箇所をリスト化
   - 優先度をつける（必須 / あれば良い）

   **FBリストの例:**
   | No | 修正箇所 | 内容 | 優先度 |
   |----|---------|------|--------|
   | 1 | トップ | キャッチコピーを変更 | 必須 |
   | 2 | About | 代表写真を差し替え | 必須 |
   | 3 | Service | 順番を入れ替え | あれば |

2. **不明点があれば確認**
   - 曖昧な指示は明確化してから修正

---

### STEP 3: 修正対応

1. **Claude Codeで修正**
   ```
   以下の修正をお願いします：
   1. トップページのキャッチコピーを「○○」に変更
   2. Aboutページの代表写真をpublic/images/ceo.jpgに差し替え
   3. ...
   ```

2. **修正後の確認**
   ```bash
   npm run dev
   ```
   - 修正箇所を目視確認
   - 他の箇所に影響がないか確認

3. **修正版をデプロイ**
   ```bash
   vercel --prod
   ```

4. **修正完了の連絡**

   **メール例:**
   ```
   件名: 【HP制作】修正完了のご連絡（株式会社○○様）

   ○○様

   ご指摘いただいた修正が完了いたしました。
   お手数ですが、再度ご確認をお願いいたします。

   ■ 確認用URL（同じ）
   https://[プロジェクト名].vercel.app/

   ■ 修正内容
   1. キャッチコピーを「○○」に変更しました
   2. 代表写真を差し替えました
   3. ...

   問題なければ、このまま納品準備に進めさせていただきます。

   よろしくお願いいたします。
   ```

---

### STEP 4: 再確認・承認

1. **承認の確認**
   - メールで承認の返信を受ける
   - または打ち合わせで口頭承認

2. **承認内容の記録**
   - 承認日
   - 承認者
   - 承認内容（メールを保存）

**複数回の修正が必要な場合:**
- STEP 2〜4を繰り返す
- 修正回数が多い場合は追加費用の相談

---

### STEP 5: ステータス更新

1. ヒアリングシートを開く
2. メニュー「**1.📋 HP制作**」→「**✏️ ステータス更新**」をクリック
3. 以下を更新：
   - **現在タスク**: 9.納品
   - **タスク保持者**: 河合（継続）
   - **状態**: 対応中
4. **「更新」**ボタンをクリック
