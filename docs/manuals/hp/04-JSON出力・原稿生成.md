# JSON出力・原稿生成マニュアル

## 概要
ヒアリングシートの内容をJSONで出力し、AIでHP原稿を生成する。

## 担当者
- 河合

## 使用ツール
- GAS（JSON出力）
- AI（原稿生成）

## フロー

### 1. JSON出力
ヒアリングシートの内容をJSONで出力する。
- GASのメニューからJSON出力機能を実行
- 出力されたJSONをコピー

### 2. AIプロンプト実行
JSON＋プロンプトでAIに原稿を生成させる。

**プロンプト構成:**
- テンプレートの構成説明
- JSONデータ
- 出力形式の指定

### 3. 原稿確認
生成された原稿を確認・修正する。
- テンプレートに沿っているか
- 内容が正確か
- 文言の調整

### 4. ステータス更新
ステータスを更新して次の工程へ。

## チェックポイント
- テンプレートに沿った原稿が生成されているか
- 内容に誤りがないか
