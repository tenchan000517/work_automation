# 企画・質問設計マニュアル

## 基本情報

| 項目 | 内容 |
|------|------|
| 商材 | ツナゲル |
| 業務内容（大項目） | インタビューショート動画 |
| 担当者 | 川崎 |
| 使用ツール | 企業カンペ |
| 成果物 | 質問案 |
| チェックポイント | 採用訴求一致 |

## 目的

採用訴求に沿ったインタビュー動画を制作するための、質問設計と企業カンペを作成する。

## シミュレーション（業務の流れ）

現地打合せ：企業担当・川崎　撮影の流れ、撮影場所の選定、カンペ作成

## 前工程

- ヒアリング内容整理完了
- 構成案作成完了

## 作業手順

### Step 1: 構成案の確認

1. ヒアリング内容整理で作成された構成案を確認
2. 採用訴求ポイントを把握
3. 動画で伝えるべきメッセージを特定

### Step 2: 質問案の作成

1. インタビュー質問を設計
2. 質問の流れを構成（導入→本題→締め）
3. 想定回答時間を考慮（ショート動画用に簡潔な回答を誘導）

**質問設計のポイント：**
- オープンクエスチョンを中心に
- 「なぜ」「どのように」を引き出す質問
- 採用ターゲットに響くエピソードを引き出す
- 回答が長くなりすぎないよう配慮

### Step 3: 企業カンペの作成

1. 質問リストを整理
2. インタビュイーの名前・役職を記載
3. 撮影時の注意事項を記載

### Step 4: 採用訴求一致チェック

- [ ] 質問内容が採用訴求ポイントと一致しているか
- [ ] ターゲット層に響く内容になっているか
- [ ] ショート動画として成立する長さを想定しているか
- [ ] 企業の差別化ポイントを引き出せる質問があるか

## 成果物

- 質問案
- 企業カンペ

## 後工程への引継ぎ

- 川崎へ現地打合せの準備として質問案・企業カンペを共有
- 撮影日程の調整

## 現地打合せフロー（川崎担当）

1. 企業担当者との打合せ
2. 撮影の流れ説明
3. 撮影場所の選定
4. カンペの最終調整
