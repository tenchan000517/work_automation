# ツナゲル 業務全体フロー

## 概要
採用支援サービス「ツナゲル」の業務フロー。採用求人原稿作成とインタビューショート動画を組み合わせた採用支援を提供。

---

## Phase 0: 受注・立ち上げ

### ① 受注・ワークス立ち上げ
- **担当**: 渡邉
- **成果物**: 企業グループ作成・企業基本情報の共有フォーマット
- **トリガー**: ワークスで担当者メンション + 必要情報フォーマット送信

↓

### ② 初回打ち合わせ日程調整
- **担当**: 渡邉
- **事前**: 先方へヒアリングシート（黄色部分）記入依頼
- **トリガー**: 日程確定報告（カレンダー更新）

↓

### ③ オンライン初回打ち合わせ
- **参加者**: 企業担当・渡邉・河合
- **内容**: 求人原稿ヒアリング、撮影日程調整（川崎と連携）
- **トリガー**: 議事録のワークスグループ共有

---

## Phase 1: ヒアリング・企画

### ④ 撮影データの共有フォルダ作成
- **担当**: 河合

↓

### ⑤ ヒアリング内容整理
- **担当**: 河合
- **ツール**: スプレッドシート
- **成果物**: 構成案
- **チェック**: 要件漏れ
- **トリガー**: 構成案をワークスで共有

↓

**【ライティング】** と **【動画制作】** に分岐

---

## Phase 2A: 原稿作成 / Phase 2B: 動画制作

| Phase 2A: 原稿作成 | Phase 2B: 動画制作 |
|---|---|
| ⑥ 原稿執筆 | ⑦ 企画・質問設計 |
| 担当：河合, 中尾文香 | 担当：川崎 |
| ツール：エンゲージ・Pairsona | ツール：企業カンペ |
| 成果物：原稿 | 成果物：質問案 |
| チェック：表現NG | チェック：採用訴求一致 |
| | ↓ |
| | ⑧ 現地打合せ（川崎） |
| | 撮影場所選定・カンペ作成 |
| | ↓ |
| | ⑨ 撮影 |
| | 担当：川崎 |
| | ツール：AI・カメラアプリ |
| | 成果物：動画素材 |
| | チェック：音声 |
| | ↓ |
| | ⑩ 編集 |
| | 担当：河合 |
| | ツール：Powerdirector |
| | 成果物：ショート動画 |
| | チェック：テロップ誤字 |

---

## Phase 3: 確認・納品

### ⑪ 原稿への動画挿入
- **Pairsona**: YouTube経由でリンク生成→原稿内に挿入
- **エンゲージ**: 動画ファイルを直接挿入

↓

### ⑫ 企業担当へ確認依頼
- **担当**: 河合
- **ツール**: メール
- **成果物**: 原稿リンク
- **チェック**: 原稿内容に相違ないか

↓

### ⑬ 修正対応（必要に応じて）

↓

### ⑭ キックオフMTG
- **担当**: 河合
- **内容**: 今後の担当者共有・スケジュール共有
- 次回FB日程の調整

---

## Phase 4: 運用・応募対応

### ⑮ 応募対応
- **担当**: 紺谷
- **ツール**: エンゲージ・Pairsona・ワークス
- **成果物**: 通話ログ・メール
- **チェック**: フロー遵守
- ※応募対応フロー使用

---

## Phase 5: 分析・フィードバック

### ⑯ 週間データ集計
- **担当**: 河合
- **ツール**: スプレッドシート・Canva
- **成果物**: スプレッドシート集計データ
- **内容**: エンゲージログイン→閲覧数・イイね数・応募数
- → 各企業ワークスグループへ共有

↓

### ⑰ 月次FB
- **担当**: 川崎, 下脇田
- **ツール**: 資料テンプレ
- **成果物**: FB資料
- **チェック**: 指標理解
- → FB議事録をFBグループにテンプレ使用で報告
- → ネクストFB日程の確定

※ 継続運用としてPhase 4-5を繰り返し
