# 打ち合わせ後対応マニュアル

## 目的

初回打ち合わせ後、撮影担当者への連絡と議事録の共有を行い、次工程へスムーズに引き継ぐ。

## 作業手順

### 1. 撮影担当者へ連絡

撮影日程確定を撮影担当者に連絡します。
撮影データの保存先フォルダも明確に指示します。

**送信手順：**
1. 連絡フォーマットから投稿文をコピー
2. 各項目を入力
3. ワークスに貼り付けて送信

**ヒアリングシート・フォルダURLの確認方法：**
企業シートのPart③（処理データ）に保存されています。

### 2. リアクション確認

ワークスのスタンプルールに従い、確認状況を把握します。

### 3. 議事録を作成

NOTTAの文字起こしをGASのプロンプトダイアログでAI用に整形し、議事録を作成します。

**手順：**
1. プロンプトダイアログを開く（メニュー「📝 プロンプト」→「議事録作成」）
2. 文字起こしを貼り付けてコピー
3. AIに貼り付けて送信
4. 出力された結果をコピー
5. 要確認箇所を修正

### 4. 議事録をワークスで共有

GASのプロンプトダイアログで議事録を投稿形式に整形し、ワークスグループに共有します。

**投稿先：** 企業のワークスグループ
**メンション：** @ALL

## トリガー（次工程へ進む条件）

議事録共有完了

## 成果物

- 撮影担当者への連絡完了
- 議事録（ワークスに共有済み）

## チェックポイント

- 撮影日程の連絡漏れ
- フォルダ作成漏れ
