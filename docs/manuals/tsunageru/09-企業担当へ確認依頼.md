# 先方確認マニュアル

## 基本情報

| 項目 | 内容 |
|------|------|
| 商材 | ツナゲル |
| 業務内容（大項目） | 作成原稿の確認 |
| 担当者 | 河合 |
| 使用ツール | メール |
| 成果物 | 原稿リンク |
| チェックポイント | 原稿内容に相違ないか |

## 目的

作成した求人原稿を企業担当者に確認してもらい、内容の承認を得る。

## シミュレーション（業務の流れ）

1. 先方へ原稿の確認
2. 修正箇所の有無の確認
3. 修正箇所の修正対応
4. キックオフの日程調整

## 前工程

- 原稿執筆完了
- 動画挿入完了（該当する場合）

## 作業手順

### Step 1: 確認依頼メールの送信

1. 原稿のリンクを準備
2. メールテンプレートを使用
3. 確認ポイントを明記
4. 回答期限を設定

### Step 2: 先方からの回答待ち

1. 期限内に回答がない場合はリマインド
2. 質問があれば対応

### Step 3: 修正対応（必要な場合）

1. 修正箇所の確認
2. 修正作業
3. 再度確認依頼

### Step 4: 承認確定

1. 最終承認を得る
2. キックオフMTGの日程調整

## 確認依頼メールテンプレート

```
件名: 【ご確認のお願い】求人原稿 - [企業名]様

[担当者名]様

お世話になっております。
Singの[担当者名]です。

この度作成いたしました求人原稿について、
ご確認をお願いいたします。

■原稿URL
[原稿リンク]

■ご確認いただきたいポイント
・会社名、担当者名の表記
・仕事内容の記載に相違がないか
・条件・待遇面の記載に相違がないか
・動画内容（該当する場合）

■回答期限
[日付]までにご回答いただけますと幸いです。

修正点やご質問等ございましたら、
お気軽にお知らせください。

何卒よろしくお願いいたします。

--
署名
```

## 修正依頼への対応フロー

修正依頼受領 → 修正内容の確認・整理 → 修正作業 → 修正完了連絡 → 再確認依頼 → 承認確定

## 成果物

- 承認済み原稿リンク

## 後工程への引継ぎ

- キックオフMTGの日程調整
- キックオフMTG担当（河合）へ引継ぎ
