# Contactページ 詳細設計

## ページ概要

| 項目 | 内容 |
|-----|------|
| 目的 | 問い合わせ・採用応募のコンバージョンを獲得する |
| ターゲット | 法人顧客（問い合わせ）、求職者（採用応募） |
| CV導線上の役割 | 行動→コンバージョン（最終段階） |
| 滞在時間目標 | 3分（フォーム入力完了まで） |
| 次のアクション | フォーム送信完了 |

---

**コンテンツ**:

- 英語表記: 「CONTACT」
- パンくず: TOP > お問い合わせ

## セクション2: 問い合わせ種別選択

### 設計意図

**マーケティング視点**:
「法人向け問い合わせ」と「採用応募」を明確に分離。ユーザーが迷わず自分に合ったフォームを選べるようにする。

**デザイン視点**:
2つの大きなボタンで選択を促す。クリックすると該当フォームにスクロール。

### レイアウト

- 構成: コンテナ幅（max-width: 800px）、2カラム横並び（SP: 縦並び）
- 高さ: auto
- 背景: #ffffff
- 余白: 上80px 下80px 左右80px（SP: 上60px 下60px 左右20px）
- ギャップ: 32px

### コンテンツ

**セクション見出し**:
```
お問い合わせ内容をお選びください
```
- フォントサイズ: PC 24px / SP 20px
- フォントウェイト: Bold (700)
- 色: #1a3a5c
- text-align: center
- margin-bottom: 40px

### 選択肢

**選択肢1: 法人向け問い合わせ**
- 背景: #f8f7f4
- padding: 40px 32px
- text-align: center
- border: 2px solid transparent
- ホバー: border-color #1a3a5c
- リンク先: #business-form
- タイトル: 「法人のお客様・ご相談」
  - フォントサイズ: 20px
  - フォントウェイト: Bold (700)
  - 色: #1a3a5c
- 説明:
  ```
  新築、リフォーム、商業施設の建設など、
  お見積もり・ご相談はこちらから
  ```
  - フォントサイズ: 14px
  - 色: #666666
  - margin-top: 12px

**選択肢2: 採用応募**
- スタイル: 同上
- リンク先: #recruit-form
- タイトル: 「採用エントリー」
- 説明:
  ```
  新卒・中途採用のエントリー、
  会社見学のお申し込みはこちらから
  ```

## セクション3: 法人向けお問い合わせフォーム

### 設計意図

**マーケティング視点**:
項目数を最小限に抑え、フォーム完了率を高める。必須項目は4つに絞り、入力ハードルを下げる。

**デザイン視点**:
フォームは清潔感のあるシンプルなデザイン。入力欄は十分な大きさを確保し、タップしやすく。

### レイアウト

- 構成: コンテナ幅（max-width: 700px）
- id: business-form
- 高さ: auto
- 背景: #f8f7f4
- 余白: 上100px 下100px 左右80px（SP: 上80px 下80px 左右20px）

### コンテンツ

**セクション見出し**:
```
法人のお客様・ご相談フォーム
```
- フォントサイズ: PC 24px / SP 20px
- フォントウェイト: Bold (700)
- 色: #1a3a5c
- margin-bottom: 16px

**リード文**:
```
以下のフォームに必要事項をご入力ください。
2営業日以内に担当者よりご連絡いたします。
```
- フォントサイズ: 15px
- 色: #666666
- margin-bottom: 40px

### フォーム項目

**項目1: お問い合わせ種別**
- ラベル: 「お問い合わせ種別」
- 必須: ○
- 入力タイプ: ラジオボタン
- 選択肢:
  - 新築のご相談
  - リフォームのご相談
  - 商業施設・工場の建設
  - 不動産管理について
  - その他

**項目2: 会社名**
- ラベル: 「会社名」
- 必須: ×
- 入力タイプ: テキスト
- プレースホルダー: 「例）株式会社○○」

**項目3: お名前**
- ラベル: 「お名前」
- 必須: ○
- 入力タイプ: テキスト
- プレースホルダー: 「例）山田 太郎」

**項目4: 電話番号**
- ラベル: 「電話番号」
- 必須: ○
- 入力タイプ: tel
- プレースホルダー: 「例）090-1234-5678」
- バリデーション: 数字とハイフンのみ

**項目5: メールアドレス**
- ラベル: 「メールアドレス」
- 必須: ○
- 入力タイプ: email
- プレースホルダー: 「例）example@email.com」
- バリデーション: メール形式

**項目6: お問い合わせ内容**
- ラベル: 「お問い合わせ内容」
- 必須: ○
- 入力タイプ: textarea
- プレースホルダー: 「ご相談内容をご記入ください」
- 行数: 6行

### フォームスタイル

- ラベル:
  - フォントサイズ: 14px
  - フォントウェイト: Medium (500)
  - 色: #333333
  - margin-bottom: 8px
- 必須マーク:
  - テキスト: 「必須」
  - 背景: #c0392b
  - 色: #ffffff
  - フォントサイズ: 11px
  - padding: 2px 6px
  - margin-left: 8px
- 入力欄:
  - 背景: #ffffff
  - border: 1px solid #d0d0d0
  - border-radius: 4px
  - padding: 14px 16px
  - フォントサイズ: 16px
  - フォーカス時: border 2px solid #1a3a5c
- 項目間: margin-bottom: 24px

**送信ボタン**:
- 文言: 「送信する」
- スタイル: Primary Large（幅100%、または中央配置で幅200px）
- margin-top: 40px

**プライバシーポリシー同意**:
```
□ プライバシーポリシーに同意する
```
- フォントサイズ: 14px
- 「プライバシーポリシー」はリンク（下線付き）
- margin-bottom: 24px

**エラー表示**:
- 色: #c0392b
- フォントサイズ: 13px
- 入力欄の直下に表示
- 例: 「電話番号を入力してください」

### レイアウト

- 構成: コンテナ幅（max-width: 700px）
- id: recruit-form
- 高さ: auto
- 背景: #ffffff
- 余白: 上100px 下100px 左右80px（SP: 上80px 下80px 左右20px）

### コンテンツ

**セクション見出し**:
```
採用エントリーフォーム
```
- フォントサイズ: PC 24px / SP 20px
- フォントウェイト: Bold (700)
- 色: #1a3a5c
- margin-bottom: 16px

**リード文**:
```
ご興味をお持ちいただきありがとうございます。
以下のフォームに必要事項をご入力ください。
「まずは話を聞いてみたい」という方も大歓迎です。
```
- フォントサイズ: 15px
- 色: #666666
- margin-bottom: 40px

### フォーム項目

**項目1: 応募種別**
- ラベル: 「応募種別」
- 必須: ○
- 入力タイプ: ラジオボタン

  - 会社見学・カジュアル面談希望

**項目2: お名前**
- ラベル: 「お名前」
- 必須: ○
- 入力タイプ: テキスト
- プレースホルダー: 「例）山田 太郎」

**項目3: ふりがな**
- ラベル: 「ふりがな」
- 必須: ○
- 入力タイプ: テキスト
- プレースホルダー: 「例）やまだ たろう」

**項目4: 電話番号**
- ラベル: 「電話番号」
- 必須: ○
- 入力タイプ: tel
- プレースホルダー: 「例）090-1234-5678」

**項目5: メールアドレス**
- ラベル: 「メールアドレス」
- 必須: ○
- 入力タイプ: email
- プレースホルダー: 「例）example@email.com」

**項目6: 学校名または現在のお勤め先**
- ラベル: 「学校名 / 現在のお勤め先」
- 必須: ×
- 入力タイプ: テキスト
- プレースホルダー: 「例）○○高校 / 株式会社○○」

**項目7: 志望動機・質問など**
- ラベル: 「志望動機・質問など」
- 必須: ×
- 入力タイプ: textarea
- プレースホルダー: 「ご質問などあればご記入ください（任意）」
- 行数: 4行

送信ボタン・プライバシーポリシー: 法人向けフォームと同様

- 高さ: auto
- 背景: #ffffff
- 余白: 上120px 下120px 左右80px（SP: 上80px 下80px 左右20px）

### コンテンツ

**完了アイコン**:
- チェックマーク（円形、#e67e22、60px）
- margin-bottom: 32px

**見出し**:
```
お問い合わせありがとうございます
```
- フォントサイズ: PC 28px / SP 22px
- フォントウェイト: Bold (700)
- 色: #1a3a5c
- text-align: center
- margin-bottom: 24px

```
お問い合わせ内容を受け付けました。
```
