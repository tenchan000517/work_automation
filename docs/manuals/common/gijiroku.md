# 議事録作成マニュアル

NOTTAの文字起こしからAIで議事録を作成する手順

## Step 1: ダイアログを開く

ヒアリングシートのメニューから議事録作成を選択します。

**HP制作の場合:**
メニュー「**2.📝 ヒアリング反映**」→「**議事録作成（プロンプトシートから）**」

## Step 2: 対象企業を選択

ドロップダウンから対象企業を選択します。

<img src="/images/hp/dialog-gijiroku.png" alt="議事録作成ダイアログ" style="max-width: 100%; border: 1px solid #ddd; border-radius: 4px; margin: 16px 0;" />

## Step 3: 文字起こしを貼り付け

NOTTAからダウンロードした文字起こしテキストを「ここに文字起こしテキストを貼り付け」欄に貼り付けます。

**文字起こしの取得方法:**
1. NOTTAにログイン
2. 該当の録音を選択
3. 「ダウンロード」→「テキスト」を選択
4. ダウンロードしたテキストファイルを開いてコピー

## Step 4: プロンプトを確認

「プレビュー」欄に生成されたプロンプトが表示されます。

**プロンプトの構成:**
- 出力フォーマット（議事録の形式）
- 注意事項（まとめ方のルール）
- 文字起こしテキスト

## Step 5: プロンプトをコピー

「**コピー**」ボタンをクリックしてプロンプトをコピーします。

## Step 6: AIに貼り付けて実行

Claude または ChatGPT を開き、コピーしたプロンプトを貼り付けて送信します。

<img src="/images/ai-gijiroku-input.png" alt="AIへの入力例" style="max-width: 100%; border: 1px solid #ddd; border-radius: 4px; margin: 16px 0;" />

## Step 7: AIの出力を確認

AIが議事録形式で出力します。

<img src="/images/ai-gijiroku-output.png" alt="AIの出力例" style="max-width: 100%; border: 1px solid #ddd; border-radius: 4px; margin: 16px 0;" />

**出力フォーマット例:**
```
■ 打ち合わせ概要
日時：
参加者：
企業名：

【企業・採用状況】
・

【求人原稿の方向性】
・

【動画制作について】
・

■ 決定事項
・撮影日程：
・撮影場所：
・インタビュー対象：
・その他：

■ 次のアクション
・
```

## Step 8: 議事録を保存・共有

AIの出力を適宜修正し、以下のいずれかで保存・共有します。

**保存先:**
- ヒアリングシートのメモ欄
- Google Driveの企業フォルダ
- ワークスグループに共有

## 議事録作成のポイント

### 良い議事録の条件

| 項目 | ポイント |
|------|---------|
| 簡潔さ | 要点を箇条書きでまとめる |
| ニュアンス | 企業の発言はそのままのニュアンスを残す |
| 不明点 | 不明点は「要確認」と記載 |
| 決定事項 | 決まったことを明確に記載 |
| 次のアクション | 誰が・何を・いつまでに |

### よくある修正ポイント

- 固有名詞（会社名、人名）の誤変換を修正
- 専門用語の誤認識を修正
- 発言者の特定が間違っている場合は修正

---

## トラブルシューティング

### 文字起こしが長すぎてAIに貼り付けられない

**解決策:**
- 文字起こしを前半・後半に分けて2回実行
- 重要な部分のみ抽出して実行

### AIの出力が期待と違う

**解決策:**
- 「もう少し詳細に」「箇条書きで」などと追加指示
- 特定のセクションを再度作成してもらう

### プロンプトシートが見つからない

**解決策:**
- メニュー「⚙️ 設定」→「📝 プロンプトシートを作成」を実行
- 「議事録作成」プロンプトが登録されているか確認
