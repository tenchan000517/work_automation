## テンプレート概要

**テンプレート名**: Local Visual

**型（Type）**: Local Business（地域密着）
**表現（Design）**: Visual Impact（ビジュアル重視）

**コンセプト**:
「地域名+業種」で検索してきた見込み客に、**写真で信頼感を与え、すぐに問い合わせさせる**。
テキストで説明するのではなく、**施工事例・現場の写真が語る**サイト。
「この会社なら安心」と一目で伝わるビジュアルインパクト。

**想定クライアント**:
- 建設業（住宅、リフォーム、外構、塗装）
- 製造業（精密加工、金属加工、木工）
- 士業・専門サービス（地域密着型）
- 「地域名+業種」で上位表示したい企業
- 写真素材（施工事例等）を持っている企業

**想定ユーザー（閲覧者）**:
- 地域住民（30-60代）: 「近くで信頼できる業者を探している」
- BtoB担当者: 「この地域で取引先を探している」
- 緊急性のあるユーザー: 「今すぐ修理/工事をお願いしたい」

**このテンプレートで達成すべきこと**:
1. 地域検索での上位表示（MEO効果向上）
2. 問い合わせ・電話の増加
3. 「この会社はちゃんとしている」という第一印象
4. 施工事例・実績を効果的に見せる

## 技術的制約（これらは変更不可）

1. **ベース**: template-fullorder からクローン
2. **データ管理**: site.json + 各ページ上部のコンテンツデータ定数
3. **スタイル**: Tailwind CSS 4.x（globals.css の @theme 内ブランドカラーのみ変更可）
4. **アニメーション**: Framer Motion 12.x
5. **画像**: next/image コンポーネント使用
6. **レスポンシブ**: PC優先、SP対応必須

## 禁止事項

1. **汎用的な表現の禁止**
   - ❌「フルワイドのヒーローセクション」
   - ✅「画面高さ100vhで、施工中の職人の背中を映した写真。右下に電話番号を大きく配置」

2. **理由なき設計の禁止**
   - ❌「CTAボタンを配置」
   - ✅「ファーストビューに電話番号を配置。理由: 緊急性の高いユーザーがすぐに電話できるように」

3. **AIデザインの典型パターン禁止**
   - ❌ 全要素にカードコンテナ
   - ❌ 紫系グラデーション
   - ❌ 全セクションで同じ余白
   - ❌ 全要素にフェードインアニメーション

4. **曖昧な指示の禁止**
   - ❌「適切な余白を設ける」
   - ✅「上余白80px、下余白40px。上を大きくする理由: 施工事例写真に視線を集中させるため」
