## テンプレート概要

**テンプレート名**: Recruit Magazine

**型（Type）**: Recruit（採用特化）
**表現（Design）**: Magazine（読ませるエディトリアル）

**コンセプト**:
求職者が「この会社で働きたい」と思うまでの心理プロセスを、**雑誌を読むように追体験させる**。
情報を羅列するのではなく、**ストーリーとして会社を理解させる**。
社員インタビューを中心に据え、「人」を通じて会社の魅力を伝える。

**想定クライアント**:
- 採用に苦戦している中小企業（製造業、建設業、サービス業）
- 人材確保が経営課題の企業
- 「うちの会社の良さが伝わらない」と感じている企業
- 採用サイトを企業サイトと別に持ちたい企業

**想定ユーザー（閲覧者）**:
- 高校新卒者（16-18歳）: 不安が大きい、先輩の姿を見たい、厳しすぎないか心配
- 中途採用者（25-40歳）: 前職との比較、成長機会、働きやすさを重視
- 保護者: 子供が働く会社の雰囲気、安定性を確認したい

**このテンプレートで達成すべきこと**:
1. 応募数の増加（現状比1.5倍以上）
2. 入社後のミスマッチ低減（離職率改善）
3. 「この会社で働きたい」という感情の喚起
4. 会社の雰囲気・人柄が伝わるサイト体験

## 技術的制約（これらは変更不可）

1. **ベース**: template-fullorder からクローン
2. **データ管理**: site.json + 各ページ上部のコンテンツデータ定数
3. **スタイル**: Tailwind CSS 4.x（globals.css の @theme 内ブランドカラーのみ変更可）
4. **アニメーション**: Framer Motion 12.x
5. **画像**: next/image コンポーネント使用
6. **レスポンシブ**: PC優先、SP対応必須

## 禁止事項

1. **汎用的な表現の禁止**
   - ❌「フルワイドのヒーローセクション」
   - ✅「画面高さ80vhで、左1/3にテキスト、右2/3に斜めにクロップした画像」

2. **理由なき設計の禁止**
   - ❌「CTAボタンを配置」
   - ✅「ファーストビューから3スクロール以内にCTAを配置。理由: ユーザーの平均離脱ポイントが4スクロール目のため」

3. **AIデザインの典型パターン禁止**
   - ❌ 全要素にカードコンテナ
   - ❌ 紫系グラデーション
   - ❌ 全セクションで同じ余白
   - ❌ 全要素にフェードインアニメーション

4. **曖昧な指示の禁止**
   - ❌「適切な余白を設ける」
   - ✅「上余白100px、下余白60px。上を大きくする理由: 前セクションとの関係を断ち切り、新しいトピックであることを示すため」
