# ステータス更新

タスク完了時に次の担当者へ引き継ぐ手順

> **重要:** 作業中のヒアリングシート（企業タブ）を開いた状態で実行してください

## 手順

### 1. メニューを開く

任意のメニューから「**✏️ ステータス更新**」を選択

<img src="/images/hp/status-menu.png" alt="ステータス更新メニュー" style="max-width: 300px; border: 1px solid #ddd; border-radius: 4px; margin: 16px 0;" />

### 2. 更新して完了

必要な項目を変更して「**更新**」をクリック

<img src="/images/hp/status-dialog.png" alt="ステータス更新ダイアログ" style="max-width: 400px; border: 1px solid #ddd; border-radius: 4px; margin: 16px 0;" />

| 項目 | 説明 |
|------|------|
| 現在タスク | 次のタスクを選択 |
| タスク保持者 | 次の担当者を選択 |
| 状態 | 対応中・確認待ち・完了など |
| メモ | 申し送り事項（任意） |

---

以上で完了です。
