# HP作成マニュアル

## 基本情報

| 項目 | 内容 |
|------|------|
| 商材 | HP制作 |
| タスク番号 | No.5 |
| 担当者 | 河合 |
| 使用ツール | Claude Code, VSCode, ターミナル |
| 成果物 | HPソースコード（Next.js） |

| 後工程 | No.6 素材撮影（オプション）または No.7 修正・根拠作成 |

## 目的

Claude Codeを使用して、テンプレートをベースにクライアント用のHPを実装する。

## 関連リソース

| リソース | URL/パス |
|---------|---------|
| HPテンプレート | https://sing-hp-template.vercel.app/ |
| 修正マニュアル | https://sing-hp-template.vercel.app/manual |
| テンプレートリポジトリ | `/mnt/c/sing-hp-template/` |

## フロー概要

```
【準備】テンプレートリポジトリをクローン/コピー
    ↓
【STEP 1】Claude Code指示文を投入
    ↓
Claude Codeが実装を進行
    ↓
【STEP 2】動作確認（ローカル）
    ↓
修正があれば指示
    ↓
【STEP 3】ビルド確認
    ↓
エラーがあれば修正
    ↓
【STEP 4】ステータス更新
```

## 作業手順

### 準備: テンプレートの準備

1. **テンプレートリポジトリをコピー**
   ```bash
   # 新しいプロジェクト用にコピー
   cp -r /mnt/c/sing-hp-template /mnt/c/[企業名]-hp
   ```

2. **ディレクトリに移動**
   ```bash
   cd /mnt/c/[企業名]-hp
   ```

3. **Git初期化（必要に応じて）**
   ```bash
   rm -rf .git
   git init
   ```

4. **依存関係のインストール**
   ```bash
   npm install
   ```

---

### STEP 1: Claude Codeで実装

1. **Claude Codeを起動**
   ```bash
   claude
   ```

2. **No.4で生成した指示文を投入**
   - 「🤖 Claude Code指示文生成」でコピーした内容を貼り付け

   ![Claude Code指示文生成ダイアログ](/images/hp/dialog-claude-code.png)

3. **Claude Codeが実装を進行**
   - ページコンポーネントの作成
   - コンテンツの埋め込み
   - スタイリングの調整

4. **追加指示（必要に応じて）**
   - 「○○ページのデザインを修正して」
   - 「フォントサイズを大きくして」
   - 「アニメーションを追加して」

---

### STEP 2: ローカルで動作確認

1. **開発サーバーを起動**
   ```bash
   npm run dev
   ```

2. **ブラウザで確認**
   - http://localhost:3000 を開く

3. **確認項目**

   **表示確認:**
   - [ ] トップページが正しく表示されるか
   - [ ] 各ページへのナビゲーションが動作するか
   - [ ] 画像が正しく表示されるか（プレースホルダー含む）
   - [ ] テキストが正しく表示されるか

   **レスポンシブ確認:**
   - [ ] PC表示（1280px以上）
   - [ ] タブレット表示（768px〜1279px）
   - [ ] スマホ表示（767px以下）
   - [ ] ハンバーガーメニューが動作するか

   **機能確認:**
   - [ ] リンクが正しく動作するか
   - [ ] CTAボタンが目立つか
   - [ ] フォームが表示されるか（送信はモック）
   - [ ] アニメーションが動作するか

4. **問題があればClaude Codeに修正指示**
   - 具体的な問題点を伝える
   - 必要に応じてスクショを共有

---

### STEP 3: ビルド確認

1. **ビルドを実行**
   ```bash
   npm run build
   ```

2. **エラーがないか確認**
   - TypeScriptエラー
   - ビルドエラー
   - 警告（Warningも確認）

3. **エラーがあればClaude Codeに修正指示**
   ```
   以下のビルドエラーを修正して：
   [エラーメッセージを貼り付け]
   ```

4. **ビルド成功を確認**
   - 「✓ Compiled successfully」が表示されればOK

---

### STEP 4: ステータス更新

1. ヒアリングシートを開く
2. メニュー「**1.📋 HP制作**」→「**✏️ ステータス更新**」をクリック
3. 以下を更新：
   - **現在タスク**:
     - 素材撮影がある場合 → 6.素材撮影
     - ない場合 → 7.修正・根拠作成
   - **タスク保持者**:
     - 素材撮影がある場合 → 川崎
     - ない場合 → 河合
   - **状態**: 対応中
4. **「更新」**ボタンをクリック

## トラブルシューティング

### npm run devでエラーが出る場合
```bash
# node_modulesを削除して再インストール
rm -rf node_modules
npm install
```

### TypeScriptエラーが多発する場合
- Claude Codeに「TypeScriptエラーを全て修正して」と指示
- エラー箇所をコピーして具体的に伝える

### 画像が表示されない場合
- public/imagesフォルダに画像があるか確認
- ファイルパスが正しいか確認（大文字小文字も）
- next/imageの設定を確認

### スタイルが崩れる場合
- Tailwindの設定（tailwind.config.js）を確認
- CSSの競合がないか確認
- 開発者ツールでスタイルを検証
