# AI構成案生成プロンプト

## 概要

ヒアリングシートの情報（Part①+Part②）を入力として、求人原稿の構成案を自動生成するプロンプト。

## 使用方法

1. GASメニュー「構成案を生成」を選択
2. ヒアリングシートから情報が自動取得される
3. 生成されたプロンプトをAI（Claude/Gemini/ChatGPT）に貼り付け
4. 出力された構成案をワークスで共有

## プロンプト（プロンプトシートのE列に入れる）

【重要な方針：差別化求人原稿の考え方】
1. 求人原稿は「選別」ではなく「理解促進」のためのもの

5. 他社との差別化は「応募させないこと」ではなく「安心して応募できる理由の言語化」

## プロンプトシートへの登録

| 列 | 内容 |
|----|------|
| A列 | 構成案生成 |
| B列 | ヒアリングシートから構成案を自動生成 |
| C列 | ヒアリングシート情報（JSON） |
| D列 | GASで取得した情報を貼り付け |
| E列 | 上記プロンプト |

## 出力後のフロー

```
構成案（AI出力）
    ↓
ワークスで共有
    ├→ 原稿チーム（河合, 中尾文香）
    │   └→ 原稿執筆
    └→ 動画チーム（川崎）
        └→ 企画・質問設計 → 撮影 → 編集
```
