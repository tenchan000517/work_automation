# Gemini用プロンプト: 採用求人ヒアリングシート Googleフォーム作成

以下の仕様に従って、Googleフォームを作成してください。

---

## フォームの基本設定

- **フォーム名**: 採用求人ヒアリングシート
- **説明文**:
```
求人原稿作成に必要な情報をご記入ください。
所要時間：約15〜20分
ご不明な点がございましたら、担当者までお問い合わせください。
```
- **回答の収集先**: 新しいスプレッドシートを作成
- **進行状況バー**: 表示する

## フォーム設定

- **回答の編集を許可**: する（送信後も編集可能）
- **回答の概要を表示**: しない
- **確認メッセージ**:
```
ご回答ありがとうございました。
内容を確認の上、担当者よりご連絡いたします。
```

---

## 注意事項

- 各セクションは「次のセクションに進む」で区切ってください
- 必須項目（○マーク）は「必須」設定をオンにしてください
- ヘルプテキストは各質問の下に表示される補足説明です
- 入力形式は指定通りに設定してください（特にラジオボタンとチェックボックスの区別）
